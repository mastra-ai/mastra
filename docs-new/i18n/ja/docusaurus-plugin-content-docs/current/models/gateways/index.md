# ゲートウェイプロバイダー \{#gateway-providers\}

ゲートウェイプロバイダーは、複数のモデルプロバイダーをまとめ、キャッシュ、レート制限、分析、自動フェイルオーバーなどの機能を提供します。可観測性の確保やコスト管理、複数プロバイダーへのアクセスの簡素化が必要な場合に利用してください。
