This command has been deprecated and replaced by the `understand-pr` skill.

The reason: `/critique-pr` encouraged offloading critical thinking to agents without truly understanding the PR, the code areas it impacted, the architecture, or history. The `understand-pr` skill improves on this by making PR review collaborative and archaeological — it builds deep historical context, traces architecture, and works with the reviewer to form opinions grounded in evidence rather than generating review comments from a surface-level diff read.

For issue investigation, activate the `understand-issue` skill.

Please activate the `understand-pr` skill instead.
