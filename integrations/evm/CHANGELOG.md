# @mastra/evm

## 0.0.1-alpha.0

### Features

- Initial release with EVM blockchain tools for Mastra agents
- `evm-get-balance`: Query native token balance for any address on any EVM chain
- `evm-read-contract`: Read data from smart contracts using ABI-typed calls
- `evm-get-transaction`: Fetch transaction details by hash
- `evm-resolve-ens`: Resolve ENS names to addresses and reverse
- `evm-get-token-balance`: Query ERC-20 token balances
- `evm-get-block`: Get block information by number or tag
- Multi-chain support via configurable RPC URLs
