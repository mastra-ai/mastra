# Workflow.branch()

`.branch()` 方法在工作流步骤之间创建条件分支，允许根据前一步骤的结果采取不同的路径。

## 使用示例

```typescript
workflow.branch([
  [async ({ context }) => true, step1],
  [async ({ context }) => false, step2],
]);
```
