# Mastra.getGateway()

通过注册表键检索网关（将网关添加到 Mastra 时使用的键）。

## 使用示例

```typescript
import { Mastra } from '@mastra/core';

const mastra = new Mastra({
  gateways: {
    myGateway: new MyCustomGateway(),
  },
});

const gateway = mastra.getGateway('myGateway');
console.log(gateway.name); // 'My Custom Gateway'
```
