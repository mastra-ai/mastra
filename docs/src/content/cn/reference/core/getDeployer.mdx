# Mastra.getDeployer()

`.getDeployer()` 方法用于检索已在 Mastra 实例中配置的部署器实例。

## 使用示例

```typescript
mastra.getDeployer();
```

## 参数

此方法不接受任何参数。

        "配置的部署器实例；如果未配置部署器，则为 undefined。",
    },
  ]}
/>
