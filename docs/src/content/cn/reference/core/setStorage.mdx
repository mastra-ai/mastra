# Mastra.setStorage()

`.setStorage()` 方法用于为 Mastra 实例设置存储实例。此方法接受一个单一的 `MastraCompositeStore` 参数。

## 使用示例

```typescript
mastra.setStorage(
  new LibSQLStore({
    id: 'mastra-storage',
    url: ":memory:",
  }),
);
```

此方法不返回值。
