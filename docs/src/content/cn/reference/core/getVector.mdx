# Mastra.getVector()

`.getVector()` 方法用于按名称检索向量存储。该方法接受一个字符串参数作为向量存储的名称。

## 使用示例

```typescript
mastra.getVector("testVectorStore");
```

## 参数

<PropertiesTable
  content={[
    {
      name: "name",
      type: "TVectorName extends keyof TVectors",
      description:
        "要检索的向量存储名称。必须是存在于 Mastra 配置中的有效向量存储名称。",
    },
  ]}
/>

<PropertiesTable
  content={[

  ]}
