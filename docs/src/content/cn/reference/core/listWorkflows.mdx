# Mastra.listWorkflows()

`.listWorkflows()` 方法用于检索已在 Mastra 实例中配置的所有工作流。该方法接受一个可选的选项对象。

## 使用示例

```typescript
mastra.listWorkflows();
```

## 参数

<PropertiesTable
  content={[
    {
      name: "options",
      type: "{ serialized?: boolean }",
      description:
        "可选配置对象。当 `serialized` 为 true 时，返回仅包含 name 属性的简化工作流对象，而非完整的工作流实例。",
      optional: true,
    },
  ]}
/>

<PropertiesTable
  content={[
    {

    },
  ]}
/>
