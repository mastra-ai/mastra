# listScorers()

`listScorers()` 方法返回已在 Mastra 实例中注册的所有评分器。评分器用于评估 AI 输出，并可在智能体生成或工作流执行期间覆盖默认评分器。

## 使用示例

```typescript
import { mastra } from "./mastra";

// 获取所有已注册的评分器
const allScorers = mastra.listScorers();

// 访问特定的评分器
const myScorer = allScorers.relevancyScorer;
```

## 参数

此方法不需要参数。
