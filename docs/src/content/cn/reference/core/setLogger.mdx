# Mastra.setLogger()

`.setLogger()` 方法用于为 Mastra 实例中的所有组件（代理、工作流等）设置日志记录器。此方法接受一个带有 logger 属性的单一对象参数。

## 使用示例

```typescript
mastra.setLogger({ logger: new PinoLogger({ name: "testLogger" }) });
```
