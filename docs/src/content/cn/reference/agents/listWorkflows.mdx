# Agent.listWorkflows()

`.listWorkflows()` 方法检索为智能体配置的工作流，如果它们是函数则解析它们。这些工作流使智能体能够执行具有定义执行路径的复杂多步骤过程。

## 使用示例

```typescript
await agent.listWorkflows();
```
