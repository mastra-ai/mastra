# Agent.listAgents()

`.listAgents()` 方法检索为智能体配置的子智能体，如果它们是函数则解析它们。这些子智能体使智能体能够访问其他智能体并执行复杂操作。

## 使用示例

```typescript
await agent.listAgents();
```
