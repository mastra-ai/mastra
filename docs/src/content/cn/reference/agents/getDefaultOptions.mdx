# Agent.getDefaultOptions()

可以为智能体配置用于内存使用、输出格式和迭代步骤的默认选项。`.getDefaultOptions()` 方法返回这些默认值，如果它们是函数则解析它们。这些选项适用于所有 `stream()` 和 `generate()` 调用，除非被覆盖，可用于检查智能体的未知默认值。

## 使用示例

```typescript
await agent.getDefaultOptions();
```
