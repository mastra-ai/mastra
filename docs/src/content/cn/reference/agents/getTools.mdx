# Agent.getTools()

`.getTools()` 方法检索为智能体配置的工具，如果它们是函数则解析它们。这些工具扩展了智能体的能力，允许它执行特定操作或访问外部系统。

## 使用示例

```typescript
await agent.getTools();
```
