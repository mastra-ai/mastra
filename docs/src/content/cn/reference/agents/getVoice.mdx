# Agent.getVoice()

`.getVoice()` 方法检索为智能体配置的语音提供商，如果它是函数则解析它。此方法用于访问智能体的语音功能，用于文本转语音和语音转文本功能。

## 使用示例

```typescript
await agent.getVoice();
```
