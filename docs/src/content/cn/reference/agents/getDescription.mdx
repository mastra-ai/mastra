# Agent.getDescription()

`.getDescription()` 方法检索为智能体配置的描述。此方法返回描述智能体用途和能力的简单字符串描述。

## 使用示例

```typescript
agent.getDescription();
```

## 参数

此方法不带参数。

<PropertiesTable

    {

      type: "string",
      description:
        "智能体的描述，如果未配置描述则返回空字符串。",
    },
  ]}
/>
