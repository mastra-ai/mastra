# Agent.getLLM()

`.getLLM()` 方法检索为智能体配置的语言模型实例，如果它是函数则解析它。此方法提供对支持智能体功能的底层 LLM 的访问。

## 使用示例

```typescript
await agent.getLLM();
```

## 扩展使用示例

```typescript
await agent.getLLM({
  requestContext: new RequestContext(),
  model: "openai/gpt-5.1",
});
```
