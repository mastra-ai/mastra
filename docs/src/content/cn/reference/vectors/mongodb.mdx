# MongoDB 向量存储

`MongoDBVector` 类使用 [MongoDB Atlas Vector Search](https://www.mongodb.com/docs/cn/docs/atlas/atlas-vector-search/) 提供向量搜索。它可以在你的 MongoDB 集合内实现高效的相似性搜索和元数据过滤。

## 安装

```bash npm2yarn
npm install @mastra/mongodb@latest
```

## 使用示例

```typescript
import { MongoDBVector } from "@mastra/mongodb";

const store = new MongoDBVector({
  id: 'mongodb-vector',
  uri: process.env.MONGODB_URI,
  dbName: process.env.MONGODB_DATABASE,
});
```

### 自定义嵌入字段路径

如果你需要将嵌入存储在嵌套字段结构中（例如，与现有的 MongoDB 集合集成），请使用 `embeddingFieldPath` 选项：

```typescript
import { MongoDBVector } from "@mastra/mongodb";

const store = new MongoDBVector({
  id: 'mongodb-vector',
  uri: process.env.MONGODB_URI,
  dbName: process.env.MONGODB_DATABASE,
  embeddingFieldPath: 'text.contentEmbedding', // 将嵌入存储在 text.contentEmbedding
});
```

## 错误处理

存储会抛出可捕获的类型化错误：

```typescript
try {
  await store.query({
    indexName: "my_collection",
    queryVector: queryVector,
  });
} catch (error) {
  // 处理特定的错误情况
  if (error.message.includes("Invalid collection name")) {
    console.error(
      "集合名称必须以字母或下划线开头，并且只能包含有效字符。",
    );
  } } else if (error.message.includes("Collection not found")) {
    console.error("指定的集合不存在");
  } else {
    console.error("向量存储错误：", error.message);
  }
}
```

## 最佳实践

- 为过滤器中使用的元数据字段建立索引以获得最佳的查询性能。
- 在元数据中使用一致的字段命名以避免意外的查询结果。
- 定期监控索引和集合统计信息以确保高效搜索。

## 使用示例

### 使用 MongoDB 的向量嵌入

嵌入是内存的 `semanticRecall` 用来根据含义（而非关键词）检索相关消息的数值向量。

> 注意：你必须使用托管在 MongoDB Atlas 上的部署才能成功使用 MongoDB 向量数据库。

此设置使用 FastEmbed（一个本地嵌入模型）来生成向量嵌入。要使用此功能，请安装 `@mastra/fastembed`：

```bash npm2yarn
npm install @mastra/fastembed@latest
```

将以下内容添加到你的代理中：

```typescript title="src/mastra/agents/example-mongodb-agent.ts"
import { Memory } from "@mastra/memory";
import { Agent } from "@mastra/core/agent";
import { MongoDBStore, MongoDBVector } from "@mastra/mongodb";
import { fastembed } from "@mastra/fastembed";

export const mongodbAgent = new Agent({
  id: "mongodb-agent",
  name: "mongodb-agent",
  instructions:
    "你是一个能够自动回忆先前交互记忆的 AI 代理。",
  model: "openai/gpt-5.1",
  memory: new Memory({
    storage: new MongoDBStore({
      id: 'mongodb-storage',
      uri: process.env.MONGODB_URI!,
      dbName: process.env.MONGODB_DB_NAME!,
    }),
    vector: new MongoDBVector({
      id: 'mongodb-vector',
      uri: process.env.MONGODB_URI!,
      dbName: process.env.MONGODB_DB_NAME!,
    }),
    embedder: fastembed,
    options: {
      lastMessages: 10,
      semanticRecall: {
        topK: 3,
        messageRange: 2,
      },
      generateTitle: true, // 自动生成描述性的线程标题
    },
  }),
});
```
