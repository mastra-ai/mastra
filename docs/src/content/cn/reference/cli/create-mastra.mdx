# create-mastra

`create-mastra` 命令**创建**一个新的独立 Mastra 项目。使用此命令在专用目录中搭建完整的 Mastra 设置。您可以使用附加标志自定义设置过程。

## 使用方法

```bash npm2yarn
npx create-mastra@latest
```

`create-mastra` 自动以_交互_模式运行，但您也可以使用命令行参数指定项目名称和模板。

```bash npm2yarn
npx create-mastra@latest my-mastra-project -- --template coding-agent
```

查看模板的[完整列表](https://mastra.ai/api/templates.json)，并使用 `slug` 作为 `--template` CLI 标志的输入。

您还可以使用任何 GitHub 仓库作为模板（它必须是有效的 Mastra 项目）：

```bash
npx create-mastra@latest my-mastra-project -- --template mastra-ai/template-coding-agent
```

您可以在项目创建期间为特定智能体安装 Mastra 技能：

```bash npm2yarn
npx create-mastra@latest my-project -- --default --skills claude-code,cursor
```

## 遥测

默认情况下，Mastra 会收集有关您项目的匿名信息，如您的操作系统、Mastra 版本或 Node.js 版本。您可以阅读[源代码](https://github.com/mastra-ai/mastra/blob/main/packages/cli/src/analytics/index.ts)来查看收集的内容。

您可以通过设置环境变量来选择退出 CLI 分析：

```bash
MASTRA_TELEMETRY_DISABLED=1
```

您也可以在使用其他 `mastra` 命令时设置此选项：

```bash
MASTRA_TELEMETRY_DISABLED=1 npx create-mastra@latest
```
