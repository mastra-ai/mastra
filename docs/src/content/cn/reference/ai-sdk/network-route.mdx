# networkRoute()

使用 AI SDK 格式创建流式网络执行的路由处理器。此函数注册一个 HTTP `POST` 端点，接受消息，执行智能体网络，并以 AI SDK 兼容格式将响应流式传输回客户端。智能体网络允许路由智能体将任务委托给其他智能体。您必须在[自定义 API 路由](/docs/cn/docs/server/custom-api-routes)中使用它。

如果您需要框架无关的处理器，请使用 [`handleNetworkStream()`](/reference/ai-sdk/handle-network-stream)。

## 使用示例

此示例展示如何在 `/network` 端点设置网络路由，使用 ID 为 `weatherAgent` 的智能体。

```typescript title="src/mastra/index.ts"
import { Mastra } from "@mastra/core";
import { networkRoute } from "@mastra/ai-sdk";

export const mastra = new Mastra({
  server: {
    apiRoutes: [
      networkRoute({
        path: "/network",
        agent: "weatherAgent",
      }),
    ],
  },
});
```

您还可以基于 `agentId` 使用动态智能体路由。URL `/network/weatherAgent` 将解析为 ID 为 `weatherAgent` 的智能体。

```typescript title="src/mastra/index.ts"
import { Mastra } from "@mastra/core";
import { networkRoute } from "@mastra/ai-sdk";

export const mastra = new Mastra({
  server: {
    apiRoutes: [
      networkRoute({
        path: "/network/:agentId",
      }),
    ],
  },
});
```

## 附加配置

您可以使用 [`prepareSendMessagesRequest`](https://ai-sdk.dev/docs/cn/docs/reference/ai-sdk-ui/use-chat#transport.default-chat-transport.prepare-send-messages-request) 自定义发送到网络路由的请求，例如向智能体传递附加配置：

```typescript
const { error, status, sendMessage, messages, regenerate, stop } = useChat({
  transport: new DefaultChatTransport({
    api: "http://localhost:4111/network",
    prepareSendMessagesRequest({ messages }) {
      return {
        body: {
          messages,
          // 传递内存配置
          memory: {
            thread: "user-1",
            resource: "user-1",
          },
        },
      };
    },
  }),
});
```
