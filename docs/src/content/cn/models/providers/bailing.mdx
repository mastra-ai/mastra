# <img src="https://models.dev/logos/bailing.svg" alt="Bailing logo" className="inline w-8 h-8 mr-2 align-middle dark:invert dark:brightness-0 dark:contrast-200" />Bailing

通过 Mastra 的模型路由器访问 2 个 Bailing 模型。身份验证使用 `BAILING_API_TOKEN` 环境变量自动处理。

在 [Bailing 文档](https://alipaytbox.yuque.com/sxs0ba/ling/intro) 中了解更多信息。

```bash title=".env"
BAILING_API_TOKEN=your-api-key
```

```typescript title="src/mastra/agents/my-agent.ts" {7}
import { Agent } from "@mastra/core/agent";

const agent = new Agent({
  id: "my-agent",
  name: "My Agent",
  instructions: "You are a helpful assistant",
  model: "bailing/Ling-1T"
});

// 生成响应
const response = await agent.generate("Hello!");

// 流式响应
const stream = await agent.stream("Tell me a story");
for await (const chunk of stream) {
  console.log(chunk);
}
```

:::info

Mastra 使用 OpenAI 兼容的 `/chat/completions` 端点。某些提供商特定功能可能不可用。查看 [Bailing 文档](https://alipaytbox.yuque.com/sxs0ba/ling/intro) 了解详情。

:::

```typescript title="src/mastra/agents/my-agent.ts"
const agent = new Agent({
  id: "custom-agent",
  name: "custom-agent",
  model: {
    url: "https://api.tbox.cn/api/llm/v1/chat/completions",
    id: "bailing/Ling-1T",
    apiKey: process.env.BAILING_API_TOKEN,
    headers: {

});
```

### Dynamic Model Selection

```typescript title="src/mastra/agents/my-agent.ts"
const agent = new Agent({

  model: ({ requestContext }) => {
    const useAdvanced = requestContext.task === "complex";
```
