# <img src="https://models.dev/logos/iflowcn.svg" alt="iFlow logo" className="inline w-8 h-8 mr-2 align-middle dark:invert dark:brightness-0 dark:contrast-200" />iFlow

通过 Mastra 的模型路由器访问 14 个 iFlow 模型。身份验证使用 `IFLOW_API_KEY` 环境变量自动处理。

在 [iFlow 文档](https://platform.iflow.cn/en/docs) 中了解更多信息。

```bash title=".env"
IFLOW_API_KEY=your-api-key
```

```typescript title="src/mastra/agents/my-agent.ts" {7}
import { Agent } from "@mastra/core/agent";

const agent = new Agent({
  id: "my-agent",
  name: "My Agent",
  instructions: "You are a helpful assistant",
  model: "iflowcn/deepseek-r1"
});

// 生成响应
const response = await agent.generate("Hello!");

// 流式响应
const stream = await agent.stream("Tell me a story");
for await (const chunk of stream) {
  console.log(chunk);
}
```

:::info

Mastra 使用 OpenAI 兼容的 `/chat/completions` 端点。某些提供商特定功能可能不可用。查看 [iFlow 文档](https://platform.iflow.cn/en/docs) 了解详情。

:::

## 高级配置

### 自定义请求头

```typescript title="src/mastra/agents/my-agent.ts"
const agent = new Agent({
  id: "custom-agent",
  name: "custom-agent",
  model: {
    url: "https://apis.iflow.cn/v1",
    id: "iflowcn/deepseek-r1",
    apiKey: process.env.IFLOW_API_KEY,
    headers: {
      "X-Custom-Header": "value"
    }
  }
});
```

### 动态模型选择

```typescript title="src/mastra/agents/my-agent.ts"
const agent = new Agent({
  id: "dynamic-agent",
  name: "Dynamic Agent",
  model: ({ requestContext }) => {
    const useAdvanced = requestContext.task === "complex";
    return useAdvanced
      ? "iflowcn/qwen3-vl-plus"
      : "iflowcn/deepseek-r1";
  }
});
```
