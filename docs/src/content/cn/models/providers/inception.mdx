# <img src="https://models.dev/logos/inception.svg" alt="Inception logo" className="inline w-8 h-8 mr-2 align-middle dark:invert dark:brightness-0 dark:contrast-200" />Inception

通过 Mastra 的模型路由器访问 2 个 Inception 模型。身份验证使用 `INCEPTION_API_KEY` 环境变量自动处理。

在 [Inception 文档](https://platform.inceptionlabs.ai/docs) 中了解更多信息。

```bash title=".env"
INCEPTION_API_KEY=your-api-key
```

```typescript title="src/mastra/agents/my-agent.ts" {7}
import { Agent } from "@mastra/core/agent";

const agent = new Agent({
  id: "my-agent",
  name: "My Agent",
  instructions: "You are a helpful assistant",
  model: "inception/mercury"
});

// 生成响应
const response = await agent.generate("Hello!");

// 流式响应
const stream = await agent.stream("Tell me a story");
for await (const chunk of stream) {
  console.log(chunk);
}
```

:::info

Mastra 使用 OpenAI 兼容的 `/chat/completions` 端点。某些提供商特定功能可能不可用。查看 [Inception 文档](https://platform.inceptionlabs.ai/docs) 了解详情。

:::

## Advanced Configuration

### Custom Headers

```typescript title="src/mastra/agents/my-agent.ts"
const agent = new Agent({
  id: "custom-agent",
  name: "custom-agent",
  model: {
    url: "https://api.inceptionlabs.ai/v1/",
    id: "inception/mercury",
    apiKey: process.env.INCEPTION_API_KEY,
    headers: {
      "X-Custom-Header": "value"
    }
  }
});
```

### Dynamic Model Selection

```typescript title="src/mastra/agents/my-agent.ts"
const agent = new Agent({
  id: "dynamic-agent",
  name: "Dynamic Agent",
  model: ({ requestContext }) => {
    const useAdvanced = requestContext.task === "complex";
    return useAdvanced
      ? "inception/mercury-coder"
      : "inception/mercury";
  }
});
```
