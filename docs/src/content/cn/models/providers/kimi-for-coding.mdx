通过 Mastra 的模型路由器访问 2 个 Kimi For Coding 模型。身份验证使用 `KIMI_API_KEY` 环境变量自动处理。

在 [Kimi For Coding 文档](https://www.kimi.com/coding/docs/cn/docs/en/third-party-agents.html) 中了解更多信息。

```bash title=".env"
KIMI_API_KEY=your-api-key
```

```typescript title="src/mastra/agents/my-agent.ts" {7}
import { Agent } from "@mastra/core/agent";

const agent = new Agent({
  id: "my-agent",
  name: "My Agent",
  instructions: "You are a helpful assistant",
  model: "kimi-for-coding/k2p5"
});

// 生成响应
const response = await agent.generate("Hello!");

// 流式响应
const stream = await agent.stream("Tell me a story");
for await (const chunk of stream) {
  console.log(chunk);
}
```

:::info

Mastra 使用 OpenAI 兼容的 `/chat/completions` 端点。某些提供商特定功能可能不可用。查看 [Kimi For Coding 文档](https://www.kimi.com/coding/docs/cn/docs/en/third-party-agents.html) 了解详情。

:::

## 高级配置

### 自定义请求头

```typescript title="src/mastra/agents/my-agent.ts"
const agent = new Agent({
  id: "custom-agent",
  name: "custom-agent",
  model: {
    url: "https://api.kimi.com/coding/v1",
    id: "kimi-for-coding/k2p5",
    apiKey: process.env.KIMI_API_KEY,
    headers: {
      "X-Custom-Header": "value"
    }
  }
});
```

### 动态模型选择

```typescript title="src/mastra/agents/my-agent.ts"
const agent = new Agent({
  id: "dynamic-agent",
  name: "Dynamic Agent",
  model: ({ requestContext }) => {
    const useAdvanced = requestContext.task === "complex";
    return useAdvanced
      ? "kimi-for-coding/kimi-k2-thinking"
      : "kimi-for-coding/k2p5";
  }
});
```
