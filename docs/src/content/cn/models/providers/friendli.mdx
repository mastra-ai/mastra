通过 Mastra 的模型路由器访问 7 个 Friendli 模型。身份验证使用 `FRIENDLI_TOKEN` 环境变量自动处理。

在 [Friendli 文档](https://friendli.ai/docs/cn/docs/guides/serverless_endpoints/introduction) 中了解更多信息。

```bash title=".env"
FRIENDLI_TOKEN=your-api-key
```

```typescript title="src/mastra/agents/my-agent.ts" {7}
import { Agent } from "@mastra/core/agent";

const agent = new Agent({
  id: "my-agent",
  name: "My Agent",
  instructions: "You are a helpful assistant",
  model: "friendli/LGAI-EXAONE/EXAONE-4.0.1-32B"
});

// 生成响应
const response = await agent.generate("Hello!");

// 流式响应
const stream = await agent.stream("Tell me a story");
for await (const chunk of stream) {
  console.log(chunk);
}
```

:::info

Mastra 使用 OpenAI 兼容的 `/chat/completions` 端点。某些提供商特定功能可能不可用。查看 [Friendli 文档](https://friendli.ai/docs/cn/docs/guides/serverless_endpoints/introduction) 了解详情。

:::

## Advanced Configuration

### Custom Headers

```typescript title="src/mastra/agents/my-agent.ts"
const agent = new Agent({
  id: "custom-agent",
  name: "custom-agent",
  model: {
    url: "https://api.friendli.ai/serverless/v1",
    id: "friendli/LGAI-EXAONE/EXAONE-4.0.1-32B",
    apiKey: process.env.FRIENDLI_TOKEN,
    headers: {
      "X-Custom-Header": "value"
    }
  }
});
```

### Dynamic Model Selection

```typescript title="src/mastra/agents/my-agent.ts"
const agent = new Agent({
  id: "dynamic-agent",
  name: "Dynamic Agent",
  model: ({ requestContext }) => {
    const useAdvanced = requestContext.task === "complex";
    return useAdvanced
      ? "friendli/zai-org/GLM-4.7"
      : "friendli/LGAI-EXAONE/EXAONE-4.0.1-32B";
  }
});
```
