# 网关提供商

网关提供商聚合多个模型提供商，并添加缓存、速率限制、分析和自动故障转移等功能。当您需要可观测性、成本管理或简化的多提供商访问时，请使用网关。

## 自定义网关

为私有 LLM 部署或专用提供商集成创建自定义网关。请参阅[自定义网关](/models/gateways/custom-gateways)了解实现详情。
