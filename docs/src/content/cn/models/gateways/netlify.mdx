# <NetlifyLogo className="inline w-8 h-8 mr-2 align-middle" />Netlify

Netlify AI Gateway 提供对多个提供商的统一访问，内置缓存和可观测性支持。通过 Mastra 的模型路由器访问 52 个模型。

在 [Netlify 文档](https://docs.netlify.com/build/ai-gateway/overview/) 中了解更多。

## 使用方法

```typescript title="src/mastra/agents/my-agent.ts"
import { Agent } from "@mastra/core/agent";

const agent = new Agent({
  id: "my-agent",
  name: "My Agent",
  instructions: "You are a helpful assistant",
  model: "netlify/anthropic/claude-3-5-haiku-20241022"
});
```

:::info

Mastra 使用 OpenAI 兼容的 `/chat/completions` 端点。某些提供商特定功能可能不可用。详情请参阅 [Netlify 文档](https://docs.netlify.com/build/ai-gateway/overview/)。

:::

## 配置

```bash
# 使用网关 API 密钥
NETLIFY_API_KEY=your-gateway-key

# 或直接使用提供商 API 密钥
OPENAI_API_KEY=sk-...
ANTHROPIC_API_KEY=ant-...
```
