# Mastra 技能

Mastra 技能是包含指令、脚本和资源的文件夹，智能体可以发现和使用它们来获取 Mastra 知识。它们包含设置说明、最佳实践以及从 Mastra 文档获取最新信息的方法。

## 安装

要安装所有可用的 Mastra 技能，请运行以下命令：

```bash npm2yarn
npx skills add mastra-ai/skills
```

Mastra 技能适用于任何支持[技能标准](https://agentskills.io/)的编码智能体，包括 Claude Code、Cursor、Codex、OpenCode 等。

它们也可以在 [GitHub](https://github.com/mastra-ai/skills) 上获取。
