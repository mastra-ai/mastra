# Discord 社区

Discord 服务器拥有超过 1000 名成员，是 Mastra 的主要讨论论坛。Mastra 团队在北美和欧洲工作时间监控 Discord，社区成员在其他时区也很活跃。[加入 Discord 服务器](https://discord.gg/BTYqqHKUrf)。

## Discord MCP 机器人

除了社区成员之外，我们还有一个（实验性的！）Discord 机器人也可以帮助回答问题。它使用[模型上下文协议 (MCP)](/docs/cn/mcp/overview)。您可以使用 `/ask` 向它提问（公开频道或私信均可），并使用 `/cleardm`（仅限私信）清除历史记录。
