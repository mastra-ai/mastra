# 关于 Mastra

Mastra 是一个用于构建 AI 驱动的应用程序和智能体的框架，采用现代 TypeScript 技术栈。

它包含从早期原型到生产级应用程序所需的一切。Mastra 可以与 React、Next.js 和 Node 等前端和后端框架集成，也可以作为独立服务器部署到任何地方。这是构建、调优和扩展可靠 AI 产品最简单的方法。

## 为什么选择 Mastra？

专为 TypeScript 设计，围绕成熟的 AI 模式构建，Mastra 为您提供开箱即用地构建出色 AI 应用程序所需的一切。

一些主要特点包括：

- [**模型路由**](/models) - 通过一个标准接口连接 40 多个提供商。使用来自 OpenAI、Anthropic、Gemini 等的模型。

- [**智能体**](/docs/cn/agents/overview) - 构建使用大型语言模型和工具来解决开放式任务的自主智能体。智能体可以推理目标、决定使用哪些工具，并在内部迭代，直到模型发出最终答案或满足可选的停止条件。

- [**工作流**](/docs/cn/workflows/overview) - 当您需要明确控制执行时，使用 Mastra 基于图形的工作流引擎来编排复杂的多步骤流程。Mastra 工作流使用直观的控制流语法（`.then()`、`.branch()`、`.parallel()`）。

- [**人工参与**](/docs/cn/workflows/suspend-and-resume) - 暂停智能体或工作流，并在恢复之前等待用户输入或批准。Mastra 使用存储来记住执行状态，因此您可以无限期暂停并从上次停下的地方恢复。

- **上下文管理** - 在正确的时间给您的智能体提供正确的上下文。提供[消息历史记录](/docs/cn/memory/message-history)、从您的来源（API、数据库、文件）[检索](/docs/cn/rag/overview)数据，并添加类似人类的[工作](/docs/cn/memory/working-memory)和[语义](/docs/cn/memory/semantic-recall)记忆，使您的智能体行为保持一致。

- **集成** - 将智能体和工作流捆绑到现有的 React、Next.js 或 Node.js 应用程序中，或将它们作为独立端点发布。在构建 UI 时，与 Vercel 的 AI SDK UI 和 CopilotKit 等智能体库集成，让您的 AI 助手在 Web 上栩栩如生。

- **生产必备** - 交付可靠的智能体需要持续的洞察、评估和迭代。通过内置的[评分器](/docs/cn/evals/overview)和[可观察性](/docs/cn/observability/overview)，Mastra 为您提供持续观察、衡量和改进的工具。

## 您可以构建什么？

- 结合语言理解、推理和行动来解决实际任务的 AI 驱动的应用程序。

- 用于客户支持、入职或内部查询的对话式智能体。

- 用于编码、法律、金融、研究或创意工作的领域特定副驾驶。

- 触发、路由和完成多步骤流程的工作流自动化。

- 分析数据并提供可操作建议的决策支持工具。
