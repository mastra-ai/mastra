# Studio

<YouTube id="ojGu6Bi4wYk" />

Mastra Cloud 提供了托管版本的 [Studio](/docs/cn/getting-started/studio)，为您的团队提供一个共享环境来交互和测试您的智能体。团队成员可以与智能体聊天、查看跟踪、提供反馈和调整系统提示——而无需在本地运行项目。

[设置您的项目](/docs/cn/mastra-cloud/setup) 以访问云端的 Studio。

有关智能体游乐场、工作流运行器、工具测试和 MCP 服务器等功能的详细信息，请参阅 [Studio 文档](/docs/cn/getting-started/studio)。

:::note

您不需要部署智能体即可使用 Studio。Studio 在与生产部署分离的沙箱环境中运行

:::

## 共享访问权限

要邀请团队成员，请转到 [Mastra Cloud](https://cloud.mastra.ai)，点击**团队设置**，然后**成员**以添加他们。邀请后，他们可以登录并访问您项目的 Studio。

![邀请团队成员](/img/mastra-cloud/mastra-cloud-invite.png)

:::warning

团队成员可以查看环境变量和删除项目。只邀请您信任的人。

:::
