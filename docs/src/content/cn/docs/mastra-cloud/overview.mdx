# Mastra Cloud

[Mastra Cloud](https://cloud.mastra.ai) 是用于部署、管理和监控 Mastra 应用程序的平台。[导入您的项目](/docs/cn/mastra-cloud/setup) 即可开始使用。

## Studio

在云端运行 [Studio](/docs/cn/mastra-cloud/studio)，并通过链接与您的团队共享访问权限。团队成员可以测试智能体和工作流，调整系统提示，并提供反馈，而无需在本地运行项目。

## 部署

启用[部署](/docs/cn/mastra-cloud/deployment)，Mastra Cloud 将为您托管服务器。连接您的 GitHub 仓库，以便在推送到配置分支时自动部署。Mastra Cloud 将您的智能体、工具和工作流公开为 REST API 端点。

部署后，项目仪表板为您提供以下可见性：

- 部署状态和历史记录
- 构建日志和配置
- 环境变量和设置

## 可观察性

[可观察性](/docs/cn/mastra-cloud/observability) 提供对智能体和工作流在生产环境中行为方式的洞察。查看智能体运行、工作流执行和日志的详细跟踪，以调试问题并了解性能。

:::info

您不需要部署到 Mastra Cloud 即可使用可观察性。您可以从任何环境向 Mastra 发送跟踪

:::
