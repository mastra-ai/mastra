# 工作流运行器

Mastra [工作流](../workflows/overview.mdx) 可以使用内置的工作流运行器执行，也可以部署到处理编排、监控和可靠性的专门工作流执行平台。

## Inngest

Inngest 是一个用于运行后台工作流的开发者平台，无需管理基础设施。Mastra 工作流可以部署到 Inngest，它提供步骤记忆化、自动重试、实时监控和暂停/恢复功能。

访问 [Inngest 部署指南](/guides/deployment/inngest) 了解设置说明，以及 [Inngest 工作流示例](https://github.com/mastra-ai/mastra/tree/main/examples/inngest) 了解完整实现。
