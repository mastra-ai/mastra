# RunloopSandbox

Runs commands in isolated [Runloop Devboxes](https://docs.runloop.ai/docs/devboxes/overview). Uses the official [`@runloop/api-client`](https://www.npmjs.com/package/@runloop/api-client) object-oriented SDK for create, exec, suspend, resume, and shutdown.

:::info

For interface details, see [WorkspaceSandbox interface](/reference/workspace/sandbox).

:::

## Installation

```bash npm2yarn
npm install @mastra/runloop
```

## Usage

```typescript
import { Agent } from '@mastra/core/agent'
import { Workspace } from '@mastra/core/workspace'
import { RunloopSandbox } from '@mastra/runloop'

const workspace = new Workspace({
  sandbox: new RunloopSandbox({
    id: 'dev-sandbox',
    timeout: 120_000,
  }),
})

const agent = new Agent({
  name: 'dev-agent',
  model: 'anthropic/claude-opus-4-6',
  workspace,
})
```

## Workspace mounts

Mastra workspace `mount()` / `unmount()` are **not** implemented for this provider. Configure repositories, files, or storage through [Runloop devbox creation](https://docs.runloop.ai/docs/devboxes/overview) (for example blueprint, code mounts, or storage objects) in your own setup or extended options.
