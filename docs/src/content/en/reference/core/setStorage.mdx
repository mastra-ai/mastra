# Mastra.setStorage()

The `.setStorage()` method is used to set the storage instance for the Mastra instance. This method accepts a single `MastraStorage` parameter.

## Usage example

```typescript copy
mastra.setStorage(
  new LibSQLStore({
    url: ":memory:",
  }),
);
```
