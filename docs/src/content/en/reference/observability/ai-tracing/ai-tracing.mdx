# MastraAITracing

The `MastraAITracing` class provides the core AI tracing functionality for Mastra applications. It manages span creation, processing, and export to observability platforms.


## Span Types

The following span types are supported:

### Agent Operations
- `AGENT_RUN` - Agent execution from start to finish
- `LLM_GENERATION` - Individual model calls
- `TOOL_CALL` - Function/tool executions
- `MCP_TOOL_CALL` - Model Context Protocol tool executions
- `GENERIC` - Custom operations

### Workflow Operations
- `WORKFLOW_RUN` - Workflow execution
- `WORKFLOW_STEP` - Individual step processing
- `WORKFLOW_CONDITIONAL` - Conditional execution blocks
- `WORKFLOW_CONDITIONAL_EVAL` - Condition evaluations
- `WORKFLOW_PARALLEL` - Parallel execution blocks
- `WORKFLOW_LOOP` - Loop execution blocks
- `WORKFLOW_SLEEP` - Sleep/delay operations
- `WORKFLOW_WAIT_EVENT` - Event waiting operations

## Example Usage

### Basic Configuration

```typescript
import { MastraAITracing } from '@mastra/core/ai-tracing';
import { LangfuseExporter } from '@mastra/langfuse';

const tracer = new MastraAITracing({
  serviceName: 'my-service',
  sampling: {
    type: 'always'
  },
  exporters: [
    new LangfuseExporter({
      publicKey: process.env.LANGFUSE_PUBLIC_KEY,
      secretKey: process.env.LANGFUSE_SECRET_KEY,
    })
  ],
  processors: [
    new SensitiveDataFilter()
  ]
});
```

### Creating Spans

```typescript
// Start a root span
const span = tracer.startSpan({
  type: 'AGENT_RUN',
  name: 'chat-agent',
  input: { message: 'Hello' }
});

// Create a child span
const llmSpan = tracer.createChildSpan(span, {
  type: 'LLM_GENERATION',
  name: 'openai-gpt-4',
  input: { prompt: 'Respond to: Hello' }
});

// End spans
llmSpan.end({
  output: { response: 'Hi there!' },
  metadata: { tokens: 15 }
});

span.end({
  output: { finalResponse: 'Hi there!' }
});
```

### Error Handling

```typescript
const span = tracer.startSpan({
  type: 'TOOL_CALL',
  name: 'database-query'
});

try {
  // Operation that might fail
  const result = await riskyOperation();
  span.end({ output: result });
} catch (error) {
  span.error({
    error,
    metadata: {
      errorType: error.constructor.name
    }
  });
}
```
