# Hume

The Hume voice implementation in Mastra provides text-to-speech capabilities using Hume's Octave TTS API. Hume offers expressive, emotionally-aware speech synthesis.

## Usage Example

```typescript
import { HumeVoice } from '@mastra/voice-hume'

// Initialize with default configuration (uses HUME_API_KEY environment variable)
const voice = new HumeVoice()

// Initialize with custom configuration
const voice = new HumeVoice({
  speechModel: {
    apiKey: 'your-api-key',
  },
  speaker: 'your-voice-name', // Voice from Hume Voice Library or custom voice
})

// Convert text to speech
const audioStream = await voice.speak('Hello, world!', {
  speaker: 'voice-name', // Optional: override default voice
  format: { type: 'mp3' }, // Optional: 'mp3' | 'wav' | 'pcm'
  description: 'Warm and friendly tone', // Optional: delivery instructions
})
```

### listen()

This method is not supported by Hume and will throw an error. Hume does not provide speech-to-text functionality. Use `CompositeVoice` with Deepgram or another provider for STT.

## Notes

- Hume requires an API key for authentication
- Speech-to-text functionality is not supported; use `CompositeVoice` with another provider for full voice capabilities
- Hume supports voices from the [Voice Library](https://app.hume.ai/tts/voice-library) and custom voices you've created
