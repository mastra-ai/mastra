# Hume

The Hume voice implementation in Mastra provides text-to-speech (TTS) using Hume's Octave TTS API and realtime speech-to-speech via Hume's Empathic Voice Interface (EVI). Hume offers expressive, emotionally-aware speech synthesis.

## Usage Example

### Text-to-Speech

```typescript
import { HumeVoice } from '@mastra/voice-hume'

// Initialize with default configuration (uses HUME_API_KEY environment variable)
const voice = new HumeVoice()

// Initialize with custom configuration
const voice = new HumeVoice({
  speechModel: {
    apiKey: 'your-api-key',
  },
  speaker: 'your-voice-name', // Voice from Hume Voice Library or custom voice
})

// Convert text to speech
const audioStream = await voice.speak('Hello, world!', {
  speaker: 'voice-name', // Optional: override default voice
  format: { type: 'mp3' }, // Optional: 'mp3' | 'wav' | 'pcm'
  description: 'Warm and friendly tone', // Optional: delivery instructions
})
```

### Realtime Speech-to-Speech (EVI)

```typescript
import { HumeVoice } from '@mastra/voice-hume'

const voice = new HumeVoice({
  speechModel: { apiKey: process.env.HUME_API_KEY },
  speaker: 'your-voice-id',
  realtimeConfig: {
    configId: 'your-evi-config-id', // Create via Hume dashboard
    sessionSettings: {
      systemPrompt: 'You are a helpful assistant.',
      voiceId: 'your-voice-id',
    },
  },
})

await voice.connect()
voice.on('speaking', ({ audio }) => {
  // Play base64-encoded audio (WAV 48kHz)
})
voice.on('writing', ({ text, role }) => {
  console.log(`${role}: ${text}`)
})
voice.on('error', ({ message }) => console.error(message))

// Stream microphone audio
await voice.send(microphoneStream)
// Or send text for the assistant to speak
await voice.answer({ text: 'Hello, how can I help?' })

voice.close()
```

### listen()

This method is not supported by Hume and will throw an error. Hume does not provide speech-to-text functionality. Use `CompositeVoice` with Deepgram or another provider for STT.

### Realtime (EVI) Methods

#### `connect(options?)`

Establishes a WebSocket connection to Hume EVI for realtime speech-to-speech. Requires `configId` in `realtimeConfig` (constructor or `connect({ realtimeConfig: { configId } })`).

```typescript
await voice.connect()
// Or pass config at connect time
await voice.connect({ realtimeConfig: { configId: 'cfg-123' } })
```

#### `send(audioData)`

Streams audio to the EVI session. Expects PCM 16-bit audio as `Int16Array` or `NodeJS.ReadableStream`. Hume recommends \~20ms chunks for optimal transcription.

```typescript
await voice.send(microphoneStream)
await voice.send(int16ArrayChunk)
```

#### `answer(options?)`

Sends text for the assistant to speak. Triggers the assistant to respond with that text as speech.

```typescript
await voice.answer({ text: 'Hello, how can I help you today?' })
```

#### `close()`

Disconnects from the EVI WebSocket.

```typescript
voice.close()
```

#### `on(event, callback)` / `off(event, callback)`

Register or remove event listeners. Events: `speaking` (audio base64), `writing` (text + role), `error`.

```typescript
voice.on('speaking', ({ audio }) => playBase64Audio(audio))
voice.on('writing', ({ text, role }) => console.log(role, text))
voice.on('error', ({ message }) => console.error(message))
```

## Notes

- Hume requires an API key for authentication
- Speech-to-text functionality is not supported; use `CompositeVoice` with another provider for full voice capabilities
- Hume supports voices from the [Voice Library](https://app.hume.ai/tts/voice-library) and custom voices you've created
