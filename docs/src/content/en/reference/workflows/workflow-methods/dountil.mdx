# Workflow\.dountil()

The `.dountil()` method executes a step until a condition is met. It always runs the step at least once before evaluating the condition. The first time the condition is evaluated, `iterationCount` is `1`.

## Usage example

```typescript
workflow.dountil(step1, async ({ inputData }) => true)
```

## Behavior

### Iteration history

The step result preserves every iteration's output on `metadata.iterations`, alongside `metadata.iterationCount`. The step's top-level `output` continues to reflect the latest iteration.

Each entry in `metadata.iterations` contains:
