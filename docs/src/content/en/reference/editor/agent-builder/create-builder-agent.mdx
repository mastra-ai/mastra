# createBuilderAgent()

**Added in:** `@mastra/editor@0.12.1`

`createBuilderAgent()` creates the agent that powers the Agent Builder's chat-based editor. Register the returned agent on `Mastra.agents` to enable the Builder. Without it, the chat-based editor is unavailable.

It is exported from `@mastra/editor/ee`.

## Usage example

Register the builder agent on `Mastra.agents`:

```typescript title="src/mastra/index.ts"
import { Mastra } from '@mastra/core/mastra'
import { MastraEditor } from '@mastra/editor'
import { createBuilderAgent } from '@mastra/editor/ee'

export const mastra = new Mastra({
  agents: { builderAgent: createBuilderAgent() },
  editor: new MastraEditor({
    builder: {
      enabled: true,
    },
  }),
})
```

By default the agent uses the `__GATEWAY_OPENAI_MODEL__` model, which requires `OPENAI_API_KEY`. Pass a `model` override to use a different provider:

```typescript
import { createBuilderAgent } from '@mastra/editor/ee'

const builderAgent = createBuilderAgent({
  model: '__GATEWAY_ANTHROPIC_MODEL_SONNET__',
})
```
