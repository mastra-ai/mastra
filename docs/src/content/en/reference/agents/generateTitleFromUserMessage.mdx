# generateTitleFromUserMessage()

The `.generateTitleFromUserMessage()` method generates a title for a conversation thread based on a user message. This method uses the agent's language model to create a descriptive title that summarizes the conversation.

## Usage example

```typescript showLineNumbers copy
const title = await agent.generateTitleFromUserMessage("What's the weather like today?");
```
