# Agent.getMetadata()

The `.getMetadata()` method retrieves metadata configured for an agent. Use metadata to classify agents in clients without encoding that data in the agent ID or name. Metadata can be a static record or a function resolved from the request context.

## Usage example

Static metadata:

```typescript title="src/mastra/agents/support-agent.ts"
import { Agent } from '@mastra/core/agent'

export const supportAgent = new Agent({
  id: 'support-agent',
  name: 'Support Agent',
  instructions: 'You help customers with support requests.',
  model: 'openai/gpt-5.4',
  metadata: { type: 'support' },
})

const metadata = supportAgent.getMetadata()
```

Dynamic metadata resolved from the request context:

```typescript title="src/mastra/agents/support-agent.ts"
import { Agent } from '@mastra/core/agent'

export const supportAgent = new Agent({
  id: 'support-agent',
  name: 'Support Agent',
  instructions: 'You help customers with support requests.',
  model: 'openai/gpt-5.4',
  metadata: ({ requestContext }) => ({
    type: 'support',
    tenant: requestContext.get('tenant'),
  }),
})

const metadata = await supportAgent.getMetadata({ requestContext })
```
