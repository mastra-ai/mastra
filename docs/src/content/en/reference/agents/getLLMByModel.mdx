# Agent.getLLMByModel()

The `.getLLMByModel()` method retrieves a language model instance for a request-scoped model override without mutating the agent's configured model.

## Usage example

```typescript
await agent.getLLMByModel({
  model: 'openai/gpt-5.4',
})
```
