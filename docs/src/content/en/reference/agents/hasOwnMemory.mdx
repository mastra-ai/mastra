# hasOwnMemory()

The `.hasOwnMemory()` method checks whether the agent has its own memory system configured. This method returns a boolean indicating whether the agent has memory capabilities.

## Usage example

```typescript showLineNumbers copy
const hasMemory = agent.hasOwnMemory();
```

## Parameters

This method takes no parameters.
