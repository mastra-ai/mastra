# listMessagesById()

The `listMessagesById()` method retrieves specific messages by their IDs.

## Usage Example

```typescript
const storage = mastra.getStorage();
const memoryStore = await storage?.getStore('memory');

const result = await memoryStore?.listMessagesById({
  messageIds: ["msg-123", "msg-456", "msg-789"],
});
```
