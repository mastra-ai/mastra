# Workflow as a Tool


```typescript

// the tool and workflow would be shown as separate files FYI, this is just an example
const agent = new Agent({
  name: 'My Agent',
  tools: {
    processData: createTool({
      id: 'process-data',
      execute: async ({ context }) => {
        const workflow = mastra.getWorkflow('dataWorkflow');
        const run = await workflow.createRun();
        const result = await run.start({ inputData: context });
        return result.result;
      },
    })
  },
});
```
