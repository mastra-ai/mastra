# Build AI agents your users actually depend on

Mastra is a TypeScript framework that gives you everything you need to prototype fast and ship with confidence. Create your first agent with a single command and start building.

<Card>
  ## For Humans

  Run this command to create a new project

  ```bash npm2yarn
  npm create mastra@latest
  ```
</Card>

<Card>
  ## For Agents

  Installs the Mastra CLI and initializes Mastra in your app, then follows the setup instructions in your agent.

  <CopyPrompt>
    Set up Mastra in my project from scratch:

    1. Install the Mastra CLI and initialize Mastra in my app.
    1. Scaffold a first agent that uses an OpenAI model, with instructions describing a helpful assistant.
    1. Add a simple memory store and one example tool, then register the tool with the agent.
    1. Start the local dev server and send the agent a test message to verify it works.

    Follow the setup instructions printed by the CLI at each step and don't skip the verification commands.
  </CopyPrompt>
</Card>

## Explore by feature

<FeatureMatrix>
  <Feature label="Agent">
    ```ts caption="Give an LLM tools, memory, and instructions, then let it decide how to accomplish a task. Mastra is model-agnostic — swap providers by changing one string."
    import { Agent } from '@mastra/core/agent'

    export const agent = new Agent({
    	id: 'assistant',
    	name: 'Assistant',
    	instructions: 'You are a helpful assistant.',
    	model: 'MODEL_PLACEHOLDER',
    })
    ```
  </Feature>

  <Feature label="Memory">
    ```ts caption="Give your agent persistent memory so it remembers context across conversations."
    import { Agent } from '@mastra/core/agent'
    import { Memory } from '@mastra/memory'

    export const agent = new Agent({
    	id: 'assistant',
    	name: 'Assistant',
    	instructions: 'You are a helpful assistant.',
    	model: 'MODEL_PLACEHOLDER',
    	memory: new Memory({
    		options: { lastMessages: 20 },
    	}),
    })
    ```
  </Feature>

  <Feature label="Workflow">
    ```ts caption="Compose deterministic steps — including agent-backed ones — into durable, observable pipelines."
    import { Agent } from '@mastra/core/agent'
    import { createWorkflow, createStep } from '@mastra/core/workflows'
    import { z } from 'zod'

    const summarizer = new Agent({
    	id: 'summarizer',
    	name: 'Summarizer',
    	instructions: 'Summarize the input text in one sentence.',
    	model: 'MODEL_PLACEHOLDER',
    })

    const summarize = createStep({
    	id: 'summarize',
    	inputSchema: z.object({ text: z.string() }),
    	outputSchema: z.object({ summary: z.string() }),
    	execute: async ({ inputData }) => {
    		const { text } = await summarizer.generate(inputData.text)
    		return { summary: text }
    	},
    })

    export const workflow = createWorkflow({
    	id: 'pipeline',
    	inputSchema: z.object({ text: z.string() }),
    	outputSchema: z.object({ summary: z.string() }),
    })
    	.then(summarize)
    	.commit()
    ```
  </Feature>

  <Feature label="Browser">
    ```ts caption="Give an agent a real browser to navigate sites, fill forms, and extract data."
    import { Agent } from '@mastra/core/agent'
    import { AgentBrowser } from '@mastra/agent-browser'

    const browser = new AgentBrowser({ headless: false })

    export const webAgent = new Agent({
    	id: 'web-agent',
    	name: 'Web Agent',
    	instructions: 'Use browser tools to navigate websites and complete tasks.',
    	model: 'MODEL_PLACEHOLDER',
    	browser,
    })
    ```
  </Feature>

  <Feature label="Observability">
    ```ts caption="Trace every agent run, tool call, and model interaction with built-in observability."
    import { Agent } from '@mastra/core/agent'
    import { Mastra } from '@mastra/core'
    import { Observability, MastraStorageExporter } from '@mastra/observability'

    const agent = new Agent({
    	id: 'assistant',
    	name: 'Assistant',
    	instructions: 'You are a helpful assistant.',
    	model: 'MODEL_PLACEHOLDER',
    })

    export const mastra = new Mastra({
    	agents: { agent },
    	observability: new Observability({
    		configs: {
    			default: { serviceName: 'mastra', exporters: [new MastraStorageExporter()] },
    		},
    	}),
    })
    ```
  </Feature>

  <Feature label="Evals">
    ```ts caption="Score agent outputs automatically with model-graded, rule-based, and statistical scorers."
    import { Agent } from '@mastra/core/agent'
    import { createAnswerRelevancyScorer } from '@mastra/evals/scorers/prebuilt'

    export const agent = new Agent({
    	id: 'assistant',
    	name: 'Assistant',
    	instructions: 'You are a helpful assistant.',
    	model: 'MODEL_PLACEHOLDER',
    	scorers: {
    		relevancy: {
    			scorer: createAnswerRelevancyScorer({ model: 'MODEL_PLACEHOLDER' }),
    			sampling: { type: 'ratio', rate: 0.5 },
    		},
    	},
    })
    ```
  </Feature>
</FeatureMatrix>
