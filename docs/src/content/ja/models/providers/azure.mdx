Azure は AI SDK を通じて利用できます。Mastra で Azure のモデルを使用するには、プロバイダー用パッケージをインストールしてください。

プロバイダー固有の詳細なドキュメントは、[AI SDK の Azure プロバイダー ドキュメント](https://ai-sdk.dev/providers/ai-sdk-providers/azure)をご覧ください。

このプロバイダーを Mastra のエージェントで使用する方法については、[Agent Overview ドキュメント](/en/docs/agents/overview)を参照してください。

## インストール

```bash npm2yarn copy
npm install @ai-sdk/azure
```
