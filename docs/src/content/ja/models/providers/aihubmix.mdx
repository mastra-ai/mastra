AIHubMix は AI SDK を通じて利用できます。Mastra で AIHubMix のモデルを使用するには、プロバイダーのパッケージをインストールしてください。

プロバイダー固有の詳細なドキュメントは、[AI SDK の AIHubMix プロバイダー ドキュメント](https://ai-sdk.dev/providers/community-providers/aihubmix)をご覧ください。

このプロバイダーを Mastra のエージェントで使用する方法は、[Agent Overview ドキュメント](/en/docs/agents/overview)を参照してください。

## インストール

```bash npm2yarn copy
npm install @aihubmix/ai-sdk-provider
```
