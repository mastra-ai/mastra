Cloudflare Workers AI は AI SDK を通じて利用できます。Mastra でこれらのモデルを使うには、プロバイダーのパッケージをインストールしてください。

プロバイダー固有の詳細なドキュメントは、[AI SDK Cloudflare Workers AI プロバイダーのドキュメント](https://ai-sdk.dev/providers/community-providers/cloudflare-workers-ai)をご覧ください。

このプロバイダーを Mastra エージェントで使用する方法は、[Agent Overview ドキュメント](/en/docs/agents/overview)をご参照ください。

## インストール

```bash npm2yarn copy
npm install workers-ai-provider
```
