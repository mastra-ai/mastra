# Mastra.getLogsByRunId()

`.getLogsByRunId()` メソッドは、特定の実行IDとトランスポートIDに対応するログを取得するために使用します。このメソッドを利用するには、`getLogsByRunId` 操作をサポートするように設定されたロガーが必要です。

## 使い方の例

```typescript copy
mastra.getLogsByRunId({ runId: "123", transportId: "456" });
```
