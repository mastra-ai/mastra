# Mastra.getWorkflows()

`.getWorkflows()` メソッドは、Mastra インスタンスで設定されたすべてのワークフローを取得します。オプションとして options オブジェクトを渡せます。

## 使い方の例

```typescript copy
mastra.getWorkflows();
```
