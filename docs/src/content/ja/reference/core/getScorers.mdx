# getScorers()

`getScorers()` メソッドは、Mastra インスタンスに登録されているすべてのスコアラーを返します。スコアラーは AI の出力を評価するために使用され、エージェントの生成やワークフローの実行において既定のスコアラーを差し替えることができます。

## 使い方の例

```typescript
import { mastra } from './mastra';

// 登録されているすべてのスコアラーを取得
const allScorers = mastra.getScorers();

// 特定のスコアラーにアクセス
const myScorer = allScorers.relevancyScorer;
```


## パラメータ

このメソッドにはパラメータはありません。
