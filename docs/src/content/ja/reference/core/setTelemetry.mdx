# Mastra.setTelemetry()

`.setTelemetry()` メソッドは、Mastra インスタンス内のすべてのコンポーネントに対するテレメトリの設定を行うために使用します。このメソッドは、テレメトリ設定オブジェクトを1つ受け取ります。

## 使用例

```typescript copy
mastra.setTelemetry({ export: { type: "console" } });
```

## 戻り値

このメソッドは値を返しません。
