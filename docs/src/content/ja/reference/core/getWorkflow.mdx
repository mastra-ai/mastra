# Mastra.getWorkflow()

`.getWorkflow()` メソッドは、ID を指定してワークフローを取得します。このメソッドは、ワークフロー ID と任意のオプションオブジェクトを受け取ります。

## 使い方の例

```typescript copy
mastra.getWorkflow("testWorkflow");
```
