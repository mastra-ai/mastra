# Workflow.sendEvent()

`.sendEvent()` は、イベントの送信によって処理が再開されます。

## 使い方の例

```typescript copy
run.sendEvent('event-name', { value: "data" });
```

## 拡張された使用例

```typescript showLineNumbers copy
import { mastra } from "./mastra";

const run = await mastra.getWorkflow("testWorkflow").createRunAsync();

const result = run.start({
  inputData: {
    value: "こんにちは"
  }
});

setTimeout(() => {
  run.sendEvent("event-name", { value: "イベントから" });
}, 3000);
```

> この例では、ワークフローが待機状態に入る前にイベントを送信できなくなるため、`await run.start()` を直接使うのは避けてください。
