# Workflow.createRunAsync()

`.createRunAsync()` メソッドは新しいワークフロー実行インスタンスを作成し、特定の入力データでワークフローを実行できるようにします。これは `Run` インスタンスを返す現行の API です。

<Callout>
メソッドを持つオブジェクトを返すレガシーな `createRun()` メソッドについては、[Legacy Workflows](../../legacyWorkflows/createRun.mdx) セクションを参照してください。
</Callout>

## 使い方の例

```typescript copy
await workflow.createRunAsync();
```

## 拡張的な使用例

```typescript showLineNumbers copy
const workflow = mastra.getWorkflow("workflow");

const run = await workflow.createRunAsync();

const result = await run.start({
  inputData: {
    value: 10,
  },
});
```
