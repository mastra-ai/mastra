# PlayAI

Mastra の PlayAI 音声実装は、PlayAI の API を用いてテキスト読み上げ（テキスト音声合成）機能を提供します。

## 使い方の例

```typescript
import { PlayAIVoice } from "@mastra/voice-playai";

// 既定の構成で初期化（PLAYAI_API_KEY と PLAYAI_USER_ID の環境変数を使用）
const voice = new PlayAIVoice();

// 既定の構成で初期化
const voice = new PlayAIVoice({
  speechModel: {
    name: "PlayDialog",
    apiKey: process.env.PLAYAI_API_KEY,
    userId: process.env.PLAYAI_USER_ID,
  },
  speaker: "Angelo", // 既定のボイス
});

// 特定のボイスでテキストを音声化
const audioStream = await voice.speak("Hello, world!", {
  speaker:
    "s3://voice-cloning-zero-shot/b27bc13e-996f-4841-b584-4d35801aea98/original/manifest.json", // Dexter のボイス
});
```

## 手法

### speak()

設定済みの音声モデルと声を使って、テキストを音声に変換します。

### listen()

このメソッドはPlayAIではサポートされておらず、エラーを送出します。PlayAIは音声認識（speech-to-text）機能を提供していません。

## 注意事項

- PlayAI の認証には、API キーとユーザー ID の両方が必要です
- 本サービスは 2 種類のモデル「PlayDialog」と「Play3.0-mini」を提供しています
- 各ボイスには固有の S3 マニフェスト ID があり、API 呼び出し時に指定する必要があります
