# ツールAPI

ツールAPIは、Mastraプラットフォームで利用可能なツールを操作および実行するためのメソッドを提供します。



## すべてのツールの取得

利用可能なすべてのツールのリストを取得します：

```typescript
const tools = await mastraClient.getTools();
```



## 特定のツールを使用する

特定のツールのインスタンスを取得する：

```typescript
const tool = mastraClient.getTool("tool-id");
```



## ツールメソッド

### ツールの詳細を取得

ツールの詳細情報を取得します：

```typescript
const details = await tool.details();
```

### ツールの実行

特定の引数でツールを実行します：

```typescript
const result = await tool.execute({
  args: {
    param1: "value1",
    param2: "value2",
  },
  threadId: "thread-1", // Optional: Thread context
  resourceid: "resource-1", // Optional: Resource identifier
});
```
