# Agent.getTools()

`.getTools()` メソッドは、エージェントに設定されたツールを取得し、それらが関数である場合は解決（評価）します。これらのツールによってエージェントの機能が拡張され、特定のアクションの実行や外部システムへのアクセスが可能になります。

## 使い方の例

```typescript copy
await agent.getTools();
```
