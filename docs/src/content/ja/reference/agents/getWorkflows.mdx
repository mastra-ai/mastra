# Agent.getWorkflows()

`.getWorkflows()` メソッドは、エージェントに設定されたワークフローを取得し、それらが関数の場合は評価して解決します。これらのワークフローによって、エージェントは定義された実行パスに従い、複雑な複数ステップの処理を実行できます。

## 使い方の例

```typescript copy
await agent.getWorkflows();
```
