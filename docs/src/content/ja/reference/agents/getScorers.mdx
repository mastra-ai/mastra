# Agent.getScorers()

`.getScorers()` メソッドは、エージェントに設定されたスコアリング構成を取得し、それが関数の場合は評価（解決）します。このメソッドにより、エージェントの応答やパフォーマンスを評価するために使用されるスコアリングシステムにアクセスできます。

## 使い方の例

```typescript copy
await agent.getScorers();
```
