# Agent.listAgents()

`.listAgents()` メソッドは、エージェントに設定されたサブエージェントを取得し、それらが関数である場合は評価して実体化します。これらのサブエージェントにより、エージェントは他のエージェントへアクセスし、複雑な処理を実行できます。

## 使い方の例

```typescript copy
await agent.listAgents();
```
