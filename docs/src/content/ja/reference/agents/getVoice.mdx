# Agent.getVoice()

`.getVoice()` メソッドは、エージェントに設定された音声プロバイダーを取得し、それが関数であれば実体化（解決）します。テキスト読み上げ（TTS）や音声認識（STT）などの音声機能にアクセスする際に使用されます。

## 使い方の例

```typescript copy
await agent.getVoice();
```
