# Workflow.step()

`.step()` メソッドは、ワークフローに新たなステップを追加し、必要に応じてその変数や実行条件を設定します。

## 使い方

```typescript
workflow.step({
  id: "stepTwo",
  outputSchema: z.object({
    result: z.number(),
  }),
  execute: async ({ context }) => {
    return { result: 42 };
  },
});
```
