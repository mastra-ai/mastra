---
title: 'mastra lint'
description: 'Mastraプロジェクトの構文チェック'
---

`mastra lint`コマンドはMastraプロジェクトの構造とコードを検証します。

## 使用方法

```bash
mastra lint [options]
```

## オプション

| オプション | 説明 |
|--------|-------------|
| `-d, --dir <path>` | Mastraフォルダへのパス |
| `-r, --root <path>` | ルートフォルダへのパス |
| `-t, --tools <toolsDirs>` | 含めるツールファイルへのパスのカンマ区切りリスト |
