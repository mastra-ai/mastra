# ArizeExporter

AI のトレーシングデータを、Arize Phoenix、Arize AX、または OpenInference のセマンティック規約に準拠した任意の OpenTelemetry 互換のオブザーバビリティプラットフォームへ送信します。

## コンストラクタ

```typescript
new ArizeExporter(config: ArizeExporterConfig)
```


## ArizeExporterConfig

```typescript
interface ArizeExporterConfig {
  // Phoenix / OpenTelemetry の設定
  endpoint?: string;
  apiKey?: string;

  // Arize AX の設定
  spaceId?: string;

  // 共通の設定
  projectName?: string;
  headers?: Record<string, string>;

  // OtelExporterConfig から継承
  timeout?: number;
  batchSize?: number;
  logLevel?: 'debug' | 'info' | 'warn' | 'error';
  resourceAttributes?: Record<string, any>;
}
```

## 方法

### exportEvent

```typescript
async exportEvent(event: AITracingEvent): Promise<void>
```

設定されたエンドポイントにトレースイベントをエクスポートします。


### export

```typescript
async export(spans: ReadOnlyAISpan[]): Promise<void>
```

OpenTelemetry を使用し、OpenInference のセマンティック規約に従ってスパンをバッチでエクスポートします。


### シャットダウン

```typescript
async shutdown(): Promise<void>
```

未処理のデータをフラッシュし、クライアントをシャットダウンします。


## 使い方

### Phoenix の設定

```typescript
import { ArizeExporter } from '@mastra/arize';

const exporter = new ArizeExporter({
  endpoint: 'http://localhost:6006/v1/traces',
  apiKey: process.env.PHOENIX_API_KEY, // ローカルPhoenixの場合は省略可能
  projectName: 'my-ai-project',
});
```


### Arize AX の構成

```typescript
import { ArizeExporter } from '@mastra/arize';

const exporter = new ArizeExporter({
  spaceId: process.env.ARIZE_SPACE_ID!,
  apiKey: process.env.ARIZE_API_KEY!,
  projectName: 'my-ai-project',
});
```


## OpenInference セマンティック規約

ArizeExporter は、生成 AI アプリケーション向けに [OpenInference セマンティック規約](https://github.com/Arize-ai/openinference/tree/main/spec) を実装し、さまざまなオブザーバビリティプラットフォーム間で標準化されたトレース構造を提供します。
