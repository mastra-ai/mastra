# ConsoleExporter

デバッグや開発時のトレースイベントをコンソールに出力します。

## コンストラクタ

```typescript
new ConsoleExporter(logger?: IMastraLogger)
```

## 出力形式

エクスポーターはイベントの種類に応じて、異なる形式で出力します。

### SPAN&#95;STARTED

```
🚀 SPAN_STARTED
   タイプ: [span type]
   名前: [span name]
   ID: [span id]
   トレースID: [trace id]
   入力: [formatted input]
   属性: [formatted attributes]
────────────────────────────────────────
```


### SPAN&#95;ENDED

```
✅ SPAN_ENDED
   タイプ: [span type]
   名前: [span name]
   ID: [span id]
   実行時間: [duration]ms
   トレースID: [trace id]
   入力: [formatted input]
   出力: [formatted output]
   エラー: [formatted error if present]
   属性: [formatted attributes]
────────────────────────────────────────
```


### SPAN&#95;UPDATED

```
📝 SPAN_UPDATED
   タイプ: [span type]

   出力: [formatted output]
```

## 使い方

```typescript
import { ConsoleExporter } from '@mastra/core/ai-tracing';
import { ConsoleLogger, LogLevel } from '@mastra/core/logger';

// デフォルトロガーを使用（INFOレベル）
const exporter = new ConsoleExporter();

// カスタムロガーを使用
const customLogger = new ConsoleLogger({ level: LogLevel.DEBUG });
const exporterWithLogger = new ConsoleExporter(customLogger);
```


## 実装の詳細

- 属性を2スペースインデントのJSONとして整形
- スパンの継続時間をミリ秒単位で計算して表示
- シリアライズエラーを適切に処理
- 未実装のイベントタイプを警告として記録
- イベント間に80文字幅の区切り線を使用
