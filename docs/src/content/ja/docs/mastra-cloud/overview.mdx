# Mastra Cloud

[Mastra Cloud](https://mastra.ai/cloud) は、Mastra アプリケーションのデプロイ、管理、監視、デバッグを行うためのプラットフォームです。アプリケーションを[デプロイ](/docs/mastra-cloud/setting-up)すると、Mastra Cloud はエージェント、ツール、ワークフローを REST API のエンドポイントとして公開します。

<MastraCloudCallout />

## プラットフォームの機能

自動ビルド、整理されたプロジェクト、追加の設定不要で、アプリケーションのデプロイと管理が行えます。

![Platform features](/image/mastra-cloud/mastra-cloud-platform-features.jpg)

主な機能:

Mastra Cloud は、ゼロコンフィグのデプロイ、GitHub と連携した継続的インテグレーション、エージェント・ツール・ワークフローを一体としてパッケージ化するアトミックデプロイをサポートします。

## Project Dashboard

詳細な出力ログ、デプロイ状況、インタラクティブなツールを活用して、アプリケーションを監視・デバッグできます。

![Project dashboard](/image/mastra-cloud/mastra-cloud-project-dashboard.jpg)

主な機能:

Project Dashboard は、アプリケーションのステータスやデプロイの状況を俯瞰でき、ログへのアクセスやエージェント／ワークフローをテストするための組み込みプレイグラウンドも提供します。

## プロジェクト構成

適切に検出・デプロイできるよう、標準的な Mastra のプロジェクト構成を使用してください。

Mastra Cloud はリポジトリをスキャンして、次を検出します。

- **Agents**: `new Agent({...})` で定義
- **Tools**: `createTool({...})` で定義
- **Workflows**: `createWorkflow({...})` で定義
- **Steps**: `createStep({...})` で定義
- **Environment Variables**: API キーや設定変数

## 技術実装

Mastra Cloud は、Mastra のエージェント、ツール、ワークフロー向けに特化して設計されています。長時間実行のリクエストを処理し、各実行の詳細なトレースを記録し、評価（eval）を標準でサポートします。
