# Auth Overview

Mastraでは認証の処理方法を選択できるため、あなたのスタックに適したアイデンティティシステムを使用してアプリケーションのエンドポイントへのアクセスを保護できます。

シンプルな共有シークレットJWT認証から始めて、より高度なアイデンティティ機能が必要になったときにSupabase、Firebase Auth、Auth0、Clerk、WorkOSなどのプロバイダーに切り替えることができます。



## 利用可能なプロバイダー

- [JSON Web Token (JWT)](/docs/auth/jwt)
