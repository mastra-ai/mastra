# 会話履歴

会話履歴は最も基本的な種類のメモリです。現在の会話でやり取りされたメッセージの一覧です。

デフォルトでは、各リクエストに現在のメモリスレッドから直近10件のメッセージが含まれ、エージェントに短期的な会話の文脈を与えます。この上限は `lastMessages` パラメータで増やせます。

この上限は、`Memory` インスタンスに `lastMessages` パラメータを渡すことで引き上げられます。

```typescript {3-7} showLineNumbers
export const testAgent = new Agent({
  // ...
  memory: new Memory({
    options: {
      lastMessages: 20
    },
  })
});
```
