Apache License 2.0 は、ソフトウェアの使用・改変・配布に関してユーザーに幅広い権利を認める、寛容なオープンソースライセンスです。以下が可能です:

### 私たちが Apache License 2.0 を選んだ理由

私たちは、いくつかの重要な理由から Apache License 2.0 を選びました。

1. **真のオープンソース**: オープンソースの原則とコミュニティの期待に合致する、広く認知されたオープンソースライセンスです。

2. **ビジネスフレンドリー**: 商用利用や再配布に制限がなく、あらゆる規模の企業に最適です。

3. **特許保護**: 利用者に対する明確な特許保護を含み、さらなる法的安心感を提供します。

4. **コミュニティ重視**: 不要な制約なく、コミュニティによる貢献とコラボレーションを促します。

### Mastra でビジネスを築く

Apache License 2.0 は、Mastra を活用してビジネスを展開するうえで最大限の柔軟性を提供します。

#### 許可されるビジネスモデル

- **アプリケーションの構築**: Mastra を用いてアプリケーションを開発・販売する
- **コンサルティングサービスの提供**: 専門知識にもとづくアドバイザリー、実装、カスタマイズを提供する
- **カスタムソリューションの開発**: Mastra を活用し、クライアント向けに特注の AI ソリューションを構築する
- **アドオン／拡張機能の作成**: Mastra の機能を拡張する補助ツールを開発・販売する
- **トレーニングと教育**: Mastra を効果的に活用するためのコースや教材を提供する
- **ホスティング型サービス**: Mastra をホスティング／マネージドサービスとして提供する
- **SaaS プラットフォーム**: Mastra を基盤とする SaaS プラットフォームを構築する

#### 準拠した利用例

- 企業が Mastra を用いて AI 搭載のカスタマーサポートアプリを開発し、顧客企業に販売する

- 開発者が Mastra で特化型エージェントやツールを作成し、他社にライセンス提供する
- スタートアップが Mastra を活用し、業界特化のソリューション（例：医療向け AI アシスタント）を構築する
- 企業が Mastra をホスティング型サービスとして顧客に提供する
- SaaS プラットフォームが Mastra を自社の AI バックエンドとして組み込む
