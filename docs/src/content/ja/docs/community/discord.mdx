# Discord コミュニティ

Discord サーバーには 1,000 人以上のメンバーが参加しており、Mastra の主要なディスカッションフォーラムとして機能しています。Mastra チームは北米および欧州の営業時間内に Discord を常時モニタリングしており、その他のタイムゾーンでもコミュニティ メンバーが活動しています。[Discord サーバーに参加](https://discord.gg/BTYqqHKUrf)。

## Discord MCP Bot

コミュニティメンバーに加え、質問に答えるお手伝いができる（実験的な）Discordボットもあります。これは[Model Context Protocol（MCP）](/docs/agents/mcp-guide)を利用しています。公開チャンネルやDMで`/ask`を使って質問でき、DMでは`/cleardm`で履歴を消去できます。
