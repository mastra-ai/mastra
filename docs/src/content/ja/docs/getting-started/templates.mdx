# テンプレート

テンプレートは、特定のユースケースやパターンを示す、あらかじめ用意された Mastra プロジェクトです。[templates ディレクトリ](https://mastra.ai/templates)で利用可能なテンプレートを参照してください。

## テンプレートの利用

`create-mastra` コマンドでテンプレートをインストールします：

<Tabs items={["npm", "yarn", "pnpm", "bun"]}>
  <Tab>
    ```bash copy
    npx create-mastra@latest --template template-name
    ```
  </Tab>

  <Tab>
    ```bash copy
    yarn dlx create-mastra@latest --template template-name
    ```
  </Tab>

  <Tab>
    ```bash copy
    pnpm create mastra@latest --template template-name
    ```
  </Tab>

  <Tab>
    ```bash copy
    bun create mastra@latest --template template-name
    ```
  </Tab>
</Tabs>

例えば、text-to-SQL アプリを作成する場合：

```bash copy
npx create-mastra@latest --template text-to-sql
```


## テンプレートのセットアップ

インストール後:

1. **プロジェクトへ移動**:
   ```bash copy
   cd your-project-name
   ```

2. **環境変数を設定**:
   ```bash copy
   cp .env.example .env
   ```
   テンプレートのREADMEに従い、APIキーなどを`.env`に記載してください。

3. **開発を開始**:
   ```bash copy
   npm run dev
   ```

<Callout type="info">
  各テンプレートには、具体的なセットアップ手順と使用例をまとめたREADMEが付属しています。
</Callout>

テンプレートの作成に関する詳細は、[テンプレート リファレンス](/reference/templates)を参照してください。
