# チャンクサイズを調整する

大規模なドキュメントを処理する際は、各チャンクに含めるテキスト量を調整する必要が生じることがあります。既定ではチャンクは1024文字ですが、コンテンツやメモリ要件に合わせてこのサイズをカスタマイズできます。次の例では、ドキュメントを分割する際にカスタムのチャンクサイズを設定する方法を示します。

```tsx copy
import { MDocument } from "@mastra/rag";

const doc = MDocument.fromText("プレーンテキストの内容...");

const chunks = await doc.chunk({
  size: 512,
});
```
