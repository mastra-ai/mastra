# 埋め込みのアップサート

埋め込みを生成したら、ベクトル類似検索に対応したデータベースに保存する必要があります。次の例では、後で検索・取得できるよう、各種ベクトルデータベースに埋め込みを保存する方法を示します。

    `PgVector` クラスは、pgvector 拡張機能を用いて PostgreSQL にインデックスを作成し、埋め込みベクトルを挿入するためのメソッドを提供します。

    ```tsx copy
    import { openai } from "@ai-sdk/openai";
    import { PgVector } from "@mastra/pg";
    import { MDocument } from "@mastra/rag";
    import { embedMany } from "ai";

    const doc = MDocument.fromText("テキスト内容...");

    const chunks = await doc.chunk();

    const { embeddings } = await embedMany({
      values: chunks.map(chunk => chunk.text),
      model: openai.embedding("text-embedding-3-small"),
    });

    const pgVector = new PgVector({ connectionString: process.env.POSTGRES_CONNECTION_STRING! });

    await pgVector.createIndex({
      indexName: "test_index",
      dimension: 1536,
    });

    await pgVector.upsert({
      indexName: "test_index",
      vectors: embeddings,
      metadata: chunks?.map((chunk: any) => ({ text: chunk.text })),
    });
    ```

    `UpstashVector` クラスは、コレクションの作成や埋め込みの挿入を行うためのメソッドを提供し、サーバーレスのベクターデータベースである Upstash Vector に対応しています。

    ```tsx copy
    import { openai } from '@ai-sdk/openai';
    import { UpstashVector } from '@mastra/upstash';
    import { MDocument } from '@mastra/rag';
    import { embedMany } from 'ai';

    const doc = MDocument.fromText('テキストの内容...');

    const chunks = await doc.chunk();

    const { embeddings } = await embedMany({
      values: chunks.map(chunk => chunk.text),
      model: openai.embedding('text-embedding-3-small'),
    });

    const upstash = new UpstashVector({
      url: process.env.UPSTASH_URL,
      token: process.env.UPSTASH_TOKEN,
    });

    // ここでは store.createIndex は呼び出しません。Upstash では、存在しない名前空間への upsert 時にインデックス（Upstash では「名前空間」と呼ばれる）が自動作成されます。
    // その名前空間が存在しない場合、upsert によって作成されます。
    await upstash.upsert({
      indexName: 'test_collection', // Upstash の名前空間名
      vectors: embeddings,
      metadata: chunks?.map(chunk => ({ text: chunk.text })),
    });
    ```

    `LanceVectorStore` クラスは、Lance のカラムナー形式を基盤とする組み込みベクトルデータベース LanceDB に対し、テーブルやインデックスの作成や埋め込みの挿入を行うメソッドを提供します。

    ```tsx copy
    import { openai } from '@ai-sdk/openai';
    import { LanceVectorStore } from '@mastra/lance';
    import { MDocument } from '@mastra/rag';
    import { embedMany } from 'ai';

    const doc = MDocument.fromText('テキストの内容…');

    const chunks = await doc.chunk();

    const { embeddings } = await embedMany({
      values: chunks.map(chunk => chunk.text),
      model: openai.embedding('text-embedding-3-small'),
    });

    const lance = await LanceVectorStore.create('/path/to/db');

    // LanceDB では、まずテーブルを作成する必要があります
    await lance.createIndex({
      tableName: 'myVectors',
      indexName: 'vector',
      dimension: 1536,
    });

    await lance.upsert({
      tableName: 'myVectors',
      vectors: embeddings,
      metadata: chunks?.map(chunk => ({ text: chunk.text })),
    });
    ```
