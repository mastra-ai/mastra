# 順次ステップによるワークフロー

ワークフローは、特定の順序で次々と連鎖して実行することができます。

## 制御フローダイアグラム

この例では、`then` メソッドを使ってワークフローのステップを連結し、順番に実行しながらデータを受け渡す方法を示しています。

## ステップの作成

まず、ステップを作成し、ワークフローを初期化しましょう。

```ts showLineNumbers copy
import { createWorkflow, createStep } from "@mastra/core/workflows";
import { z } from "zod";

const stepOne = createStep({
  id: "stepOne",
  inputSchema: z.object({
    inputValue: z.number(),
  }),
  outputSchema: z.object({
    doubledValue: z.number(),
  }),
  execute: async ({ inputData }) => ({
    doubledValue: inputData.inputValue * 2,
  }),
});

const stepTwo = createStep({
  id: "stepTwo",
  inputSchema: z.object({
    doubledValue: z.number(),
  }),
  outputSchema: z.object({
    incrementedValue: z.number(),
  }),
  execute: async ({ inputData }) => {
    return { incrementedValue: inputData.doubledValue + 1 };
  },
});

const stepThree = createStep({
  id: "stepThree",
  inputSchema: z.object({
    incrementedValue: z.number(),
  }),
  outputSchema: z.object({
    tripledValue: z.number(),
  }),
  execute: async ({ inputData }) => {
    return { tripledValue: inputData.incrementedValue * 3 };
  },
});

const myWorkflow = createWorkflow({
  id: "my-workflow",
  inputSchema: z.object({
    inputValue: z.number(),
  }),
  outputSchema: z.object({
    tripledValue: z.number(),
  }),
});
```

## ステップを連結してワークフローを実行する

それでは、ステップを順番につなげてみましょう。

```ts showLineNumbers copy
// sequential steps
myWorkflow.step(stepOne).then(stepTwo).then(stepThree);

myWorkflow.commit();

const { start } = myWorkflow.createRun();

const res = await start({ inputData: { inputValue: 90 } });
```
