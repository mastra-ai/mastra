Welcome to Lesson 3. This lesson connects what you ran in Studio to the code.

You'll open the project files and walk through the core building blocks you'll use throughout the course. First, the agent file, where you define behavior, instructions, model choice, tools, and memory. Then the tools themselves, plain TypeScript functions with validated inputs and structured outputs. Finally, workflows, where explicit multi-step logic is defined and agents can be called as part of a larger process.

By the end, you'll understand how agents, tools, and workflows fit together and why each exists. Next, you'll begin creating your own Theme Park Companion Agent and start attaching real capabilities to it.
