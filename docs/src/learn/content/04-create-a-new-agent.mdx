Welcome to Lesson 4. In this lesson you create your first real agent.

You'll add the Theme Park Agent to the project, define its instructions, register it in the app, and run it inside Mastra Studio. The goal is simple for now: shape its behavior and responses before adding any external capabilities.

You'll also enable memory and see how it changes follow-up conversations from one-off replies to a continuous interaction. By the end, you'll have a working agent you can test, tweak, and debug, ready to gain real abilities when we start attaching tools next.

> _"An agent without tools can still be useful — it just can't do anything. Start with the instructions, get the voice right, then give it capabilities."_
>
> — Guil
