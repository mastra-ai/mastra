Welcome to Lesson 2. This lesson gets your environment ready.

You'll create a new Mastra project using the CLI, connect a model provider, and understand the setup options you're choosing during initialization. Then you'll run the project locally and open Mastra Studio, where you can interact with agents, test tools in isolation, execute workflows step by step, and inspect traces.

By the end, you'll have a working project on your machine and a clear view of the development workflow you'll use throughout the course. Up next, we map what you saw in Studio back to the code so you know exactly where each piece lives.

> _"Getting the first run working is the hardest part. Once you see an agent respond in Studio, everything else clicks."_
