# Dev

## `mastra dev`

Starts the Mastra local development server that exposes REST endpoints for your agents, tools and workflows.
It also serves the playground web interface.

### Flags

- `-d, --dir <dir>`: Path to your mastra folder
- `-e, --env <env>`: Environment File to use (defaults to .env.development)
- `-t, --tools <toolsDirs>`: Comma-separated list of paths to tool files to include
