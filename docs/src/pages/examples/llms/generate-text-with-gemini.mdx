# Generate Text with Gemini

This example demonstrates how to interact with Googleâ€™s Gemini language model. After providing a prompt, the Gemini model returns a text response.

```ts showLineNumbers copy
import { Mastra } from '@mastra/core';

const mastra = new Mastra();

const llm = mastra.LLM({
  provider: 'GOOGLE',
  name: 'gemini-1.5-flash',
  apiKey: process.env.GEMINI_API_KEY,
});

const result = await llm.generate('Who invented the submarine?');

console.log(result.text);
```
