# Generate Text

This example demonstrates how to interact with OpenAI&apos;s GPT-4 model for basic text generation. When provided a prompt, the GPT-4 returns a text response.

```ts showLineNumbers copy
import { Mastra } from '@mastra/core';

const mastra = new Mastra();

const llm = mastra.LLM({
  provider: 'OPEN_AI',
  name: 'gpt-4o',
});

const response = await llm.generate('What is a wormhole?');

console.log(response.text);
```
