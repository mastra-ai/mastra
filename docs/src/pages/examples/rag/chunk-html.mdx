# Chunk HTML

This example shows you how to chunk HTML content. By using the `chunk` method, you can split the HTML into smaller pieces, which can be used for various purposes, such as search or retrieval.

```tsx copy
import { MDocument } from "@mastra/rag";

const doc = MDocument.fromHTML("<p>Your HTML content...</p>");

const chunks = await doc.chunk();
```
