# Semantically Chunking HTML

HTML content requires segmentation into smaller pieces while maintaining document structure. The chunk method segments HTML content by preserving HTML tag structure and element hierarchy. This example demonstrates HTML document chunking for search and retrieval.

```tsx copy
import { MDocument } from "@mastra/rag";

const html = `
<div>
    <h1>h1 content...</h1>
    <p>p content...</p>
</div>
`;

const doc = MDocument.fromHTML(html);

const chunks = await doc.chunk({
  headers: [
    ["h1", "Header 1"],
    ["p", "Paragraph"],
  ],
});

console.log(chunks);
```
