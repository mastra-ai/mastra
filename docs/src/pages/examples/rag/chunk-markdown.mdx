# Chunk Markdown

Markdown uses a compact syntax compared to HTML. The `chunk` method segments Markdown content by preserving structural elements such as headers, lists, and code blocks. This example demonstrates Markdown document chunking for search and retrieval.

```tsx copy
import { MDocument } from "@mastra/rag";

const doc = MDocument.fromMarkdown("# Your markdown content...");

const chunks = await doc.chunk();
```
