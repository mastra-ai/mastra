# Semantically Chunking JSON

JSON data requires segmentation while preserving object structure. The chunk method segments JSON content by maintaining key-value relationships and nested object hierarchies. This example demonstrates JSON document chunking for search and retrieval.

```tsx copy
import { MDocument } from "@mastra/rag";

const testJson = {
  name: "John Doe",
  age: 30,
  email: "john.doe@example.com",
};

const doc = MDocument.fromJSON(JSON.stringify(testJson));

const chunks = await doc.chunk({
  maxSize: 100,
});

console.log(chunks);
```
