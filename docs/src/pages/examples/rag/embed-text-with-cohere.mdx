# Embed Text with Cohere

This example shows you how to embed text with Cohere. By using the `embed` method, you can embed the text into a vector database, which can be used for various purposes, such as search or retrieval.

```tsx copy
import { MDocument, embed } from "@mastra/rag";

const doc = MDocument.fromText("Your text content...");

const chunks = await doc.chunk();

const { embeddings } = await embed(chunks, {
  provider: "COHERE",
  model: "embed-english-v3.0",
});
```
