# <NetlifyLogo className="inline w-8 h-8 mr-2 align-middle" />Netlify

Netlify AI Gateway は、キャッシュとオブザーバビリティを備え、複数のプロバイダーへの一元的なアクセスを提供します。Mastra のモデルルーターを通じて 34 のモデルにアクセスできます。

詳しくは [Netlify ドキュメント](https://docs.netlify.com/build/ai-gateway/overview/)をご覧ください。

<div id="usage">
  ## 使い方
</div>

```typescript
import { Agent } from "@mastra/core";

const agent = new Agent({
  name: "my-agent",
  instructions: "あなたは親切なアシスタントです",
  model: "netlify/anthropic/claude-3-5-haiku-20241022"
});
```

:::info

Mastra は OpenAI 互換の `/chat/completions` エンドポイントを使用します。プロバイダー固有の機能の一部は利用できない場合があります。詳細は [Netlify のドキュメント](https://docs.netlify.com/build/ai-gateway/overview/)をご確認ください。

:::


<div id="configuration">
  ## 設定
</div>

```bash
# ゲートウェイAPIキーを使用
NETLIFY_API_KEY=your-gateway-key

# またはプロバイダーのAPIキーを直接使用
OPENAI_API_KEY=sk-...
ANTHROPIC_API_KEY=ant-...
```
