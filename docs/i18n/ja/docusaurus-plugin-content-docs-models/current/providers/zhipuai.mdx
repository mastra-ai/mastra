# <img src="https://models.dev/logos/zhipuai.svg" alt="Zhipu AI logo" className="inline w-8 h-8 mr-2 align-middle dark:invert dark:brightness-0 dark:contrast-200" />Zhipu AI

Mastraのモデルルーター経由で、5つのZhipu AIモデルにアクセスできます。認証は `ZHIPU_API_KEY` 環境変数を用いて自動的に行われます。

詳しくは [Zhipu AIのドキュメント](https://docs.z.ai/guides/overview/pricing) をご覧ください。

```bash
ZHIPU_API_KEY=your-api-key
```

```typescript
import { Agent } from "@mastra/core";

const agent = new Agent({
  name: "my-agent",
  instructions: "あなたは親切なアシスタントです",
  model: "zhipuai/glm-4.5"
});

// レスポンスを生成
const response = await agent.generate("Hello!");

// レスポンスをストリーム
const stream = await agent.stream("物語を聞かせて");
for await (const chunk of stream) {
  console.log(chunk);
}
```

:::info

Mastra は OpenAI 互換の `/chat/completions` エンドポイントを使用します。プロバイダー固有の機能の一部は利用できない場合があります。詳細は [Zhipu AI のドキュメント](https://docs.z.ai/guides/overview/pricing)をご確認ください。

:::
