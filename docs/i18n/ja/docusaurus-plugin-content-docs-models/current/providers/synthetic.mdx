# <img src="https://models.dev/logos/synthetic.svg" alt="Synthetic logo" className="inline w-8 h-8 mr-2 align-middle dark:invert dark:brightness-0 dark:contrast-200" />Synthetic

Mastraのモデルルーター経由で、22のSyntheticモデルにアクセスできます。認証は `SYNTHETIC_API_KEY` 環境変数により自動的に行われます。

詳しくは [Syntheticのドキュメント](https://synthetic.new/pricing) をご覧ください。

```bash
SYNTHETIC_API_KEY=your-api-key
```

```typescript
import { Agent } from "@mastra/core";

const agent = new Agent({
  name: "my-agent",
  instructions: "あなたは親切なアシスタントです",
  model: "synthetic/hf:MiniMaxAI/MiniMax-M2"
});

// レスポンスを生成
const response = await agent.generate("Hello!");

// レスポンスをストリーミング
const stream = await agent.stream("物語を聞かせて");
for await (const chunk of stream) {
  console.log(chunk);
}
```

:::info

Mastra は OpenAI 互換の `/chat/completions` エンドポイントを使用します。プロバイダー独自の機能の一部は利用できない場合があります。詳しくは [Synthetic のドキュメント](https://synthetic.new/pricing)をご確認ください。

:::

const agent = new Agent({
  name: "dynamic-agent",
  model: ({ runtimeContext }) => {
    const useAdvanced = runtimeContext.task === "complex";
    return useAdvanced
      ? "synthetic/hf:zai-org/GLM-4.6"
      : "synthetic/hf:MiniMaxAI/MiniMax-M2";
  }
});
