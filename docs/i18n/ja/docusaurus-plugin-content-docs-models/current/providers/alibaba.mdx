Mastraのモデルルーターを通じて、Alibabaの39のモデルにアクセスできます。認証は、`DASHSCOPE_API_KEY` 環境変数を使用して自動的に行われます。

詳しくは[Alibabaのドキュメント](https://www.alibabacloud.com/help/en/model-studio/models)をご覧ください。

```bash
DASHSCOPE_API_KEY=your-api-key
```

```typescript
import { Agent } from "@mastra/core";

const agent = new Agent({
  name: "my-agent",
  instructions: "あなたは親切なアシスタントです",
  model: "alibaba/qvq-max"
});

// レスポンスを生成
const response = await agent.generate("Hello!");

// レスポンスをストリーミング
const stream = await agent.stream("物語を聞かせて");
for await (const chunk of stream) {
  console.log(chunk);
}
```

:::info

Mastra は OpenAI 互換の `/chat/completions` エンドポイントを使用します。プロバイダー固有の機能の一部は利用できない場合があります。詳しくは [Alibaba のドキュメント](https://www.alibabacloud.com/help/en/model-studio/models)をご確認ください。
