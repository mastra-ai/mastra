Azure は AI SDK を通じて利用できます。Mastra で同社のモデルを使用するには、プロバイダーのパッケージをインストールしてください。

プロバイダー固有の詳細なドキュメントは、[AI SDK の Azure プロバイダー ドキュメント](https://ai-sdk.dev/providers/ai-sdk-providers/azure)をご参照ください。

このプロバイダーを Mastra のエージェントで使用する方法については、[Agent Overview ドキュメント](/docs/agents/overview)をご覧ください。
