AIHubMix は AI SDK を通じてご利用いただけます。Mastra でこれらのモデルを使用するには、プロバイダー用パッケージをインストールしてください。

プロバイダー固有の詳細なドキュメントは、[AI SDK の AIHubMix プロバイダードキュメント](https://ai-sdk.dev/providers/community-providers/aihubmix)をご覧ください。

このプロバイダーを Mastra のエージェントで使用する場合は、[エージェント概要ドキュメント](/docs/agents/overview)をご参照ください。
