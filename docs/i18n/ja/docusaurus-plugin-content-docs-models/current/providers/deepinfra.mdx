# <img src="https://models.dev/logos/deepinfra.svg" alt="Deep Infra logo" className="inline w-8 h-8 mr-2 align-middle dark:invert dark:brightness-0 dark:contrast-200" />Deep Infra

Mastraのモデルルーターを通じて、4つのDeep Infraモデルにアクセスできます。認証は `DEEPINFRA_API_KEY` 環境変数によって自動的に行われます。

詳しくは[Deep Infraのドキュメント](https://deepinfra.com/models)をご覧ください。

```bash
DEEPINFRA_API_KEY=your-api-key
```

```typescript
import { Agent } from "@mastra/core";

const agent = new Agent({
  name: "my-agent",
  instructions: "あなたは親切なアシスタントです",
  model: "deepinfra/Qwen/Qwen3-Coder-480B-A35B-Instruct"
});

// レスポンスを生成
const response = await agent.generate("Hello!");

// レスポンスをストリーム
const stream = await agent.stream("物語を聞かせて");
for await (const chunk of stream) {
  console.log(chunk);
}
```

:::info

Mastra は OpenAI 互換の `/chat/completions` エンドポイントを使用します。プロバイダー固有の機能の一部は利用できない場合があります。詳しくは [Deep Infra のドキュメント](https://deepinfra.com/models)をご確認ください。

:::

<div id="direct-provider-installation">
  ## プロバイダーを直接インストールする
</div>

このプロバイダーはスタンドアロンのパッケージとして直接インストールでき、Mastra のモデルルーター文字列の代わりに使用できます。詳しくは[パッケージのドキュメント](https://www.npmjs.com/package/@ai-sdk/deepinfra)をご覧ください。

```bash npm2yarn copy
npm install @ai-sdk/deepinfra
```

各プロバイダー向けの詳細なドキュメントは、[AI SDK の DeepInfra プロバイダー ドキュメント](https://ai-sdk.dev/providers/ai-sdk-providers/deepinfra)をご覧ください。
