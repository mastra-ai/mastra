<div id="speechify">
  # Speechify
</div>

Mastra における Speechify の音声機能実装は、Speechify の API を利用してテキスト読み上げ（TTS）機能を提供します。

<div id="usage-example">
  ## 使い方の例
</div>

```typescript
import { SpeechifyVoice } from "@mastra/voice-speechify";

// 既定の設定で初期化（SPEECHIFY_API_KEY 環境変数を使用）
const voice = new SpeechifyVoice();

// カスタム設定で初期化
const voice = new SpeechifyVoice({
  speechModel: {
    name: "simba-english",
    apiKey: "your-api-key",
  },
  speaker: "george", // 既定の音声
});

// テキスト読み上げ（TTS）
const audioStream = await voice.speak("Hello, world!", {
  speaker: "henry", // 既定の音声を上書き
});
```

<div id="listen">
  ### listen()
</div>

このメソッドはSpeechifyではサポートされておらず、エラーを発生させます。Speechifyは音声認識（speech-to-text）機能を提供していません。

<div id="notes">
  ## 注意事項
</div>

- Speechify の認証には API キーが必要です
- 既定のモデルは「simba-english」です
- 音声認識（speech-to-text）機能には対応していません
- 追加の音声ストリームオプションは、speak() メソッドの options パラメータで指定できます
