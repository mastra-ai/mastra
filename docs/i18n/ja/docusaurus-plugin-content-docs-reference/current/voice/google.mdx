# Google

Mastra の Google Voice 実装は、Google Cloud サービスを利用して、テキスト読み上げ（TTS）と音声文字起こし（STT）の両方に対応しています。複数の音声、言語、および高度なオーディオ設定オプションをサポートします。

## 使い方の例

```typescript
import { GoogleVoice } from "@mastra/voice-google";

// 既定の設定で初期化（GOOGLE_API_KEY 環境変数を使用）
const voice = new GoogleVoice();

// カスタム設定で初期化
const voice = new GoogleVoice({
  speechModel: {
    apiKey: "your-speech-api-key",
  },
  listeningModel: {
    apiKey: "your-listening-api-key",
  },
  speaker: "en-US-Casual-K",
});

// テキスト読み上げ（Text-to-Speech）
const audioStream = await voice.speak("Hello, world!", {
  languageCode: "en-US",
  audioConfig: {
    audioEncoding: "LINEAR16",
  },
});

// 音声認識（Speech-to-Text）
const transcript = await voice.listen(audioStream, {
  config: {
    encoding: "LINEAR16",
    languageCode: "en-US",
  },
});

// 指定した言語で利用可能な話者一覧を取得
const voices = await voice.getSpeakers({ languageCode: "en-US" });
```

## 重要な注意事項

1. Google Cloud の API キーが必要です。`GOOGLE_API_KEY` 環境変数で設定するか、コンストラクターに渡してください。
2. 既定の音声は「en-US-Casual-K」に設定されています。
3. テキスト読み上げ（text-to-speech）と音声認識（speech-to-text）の両サービスは、既定のオーディオエンコーディングとして LINEAR16 を使用します。
4. `speak()` メソッドは、Google Cloud Text-to-Speech API を通じて高度な音声設定をサポートします。
5. `listen()` メソッドは、Google Cloud Speech-to-Text API を通じて多様な認識設定をサポートします。
6. 利用可能な音声は、`getSpeakers()` メソッドで言語コードによる絞り込みが可能です。
