<div id="azure">
  # Azure
</div>

Mastra の AzureVoice クラスは、Microsoft Azure Cognitive Services を利用して、テキスト読み上げと音声認識の機能を提供します。

<div id="usage-example">
  ## 使い方の例
</div>

```typescript
import { AzureVoice } from "@mastra/voice-azure";

// 設定で初期化する
const voice = new AzureVoice({
  speechModel: {
    name: "neural",
    apiKey: "your-azure-speech-api-key",
    region: "eastus",
  },
  listeningModel: {
    name: "whisper",
    apiKey: "your-azure-speech-api-key",
    region: "eastus",
  },
  speaker: "en-US-JennyNeural", // 既定の音声
});

// テキストを音声に変換する
const audioStream = await voice.speak("Hello, how can I help you?", {
  speaker: "en-US-GuyNeural", // 既定の音声を上書きする
  style: "cheerful", // 音声スタイル
});

// 音声をテキストに変換する
const text = await voice.listen(audioStream, {
  filetype: "wav",
  language: "en-US",
});
```

<div id="notes">
  ## メモ
</div>

- API キーはコンストラクターのオプションまたは環境変数（AZURE_SPEECH_KEY、AZURE_SPEECH_REGION）で指定できます
- Azure は多言語に対応した幅広いニューラル音声を提供しています
- 一部の音声は cheerful、sad、angry などの話法スタイルに対応しています
- 音声認識は複数の音声形式と言語をサポートします
- Azure の音声サービスは、自然な話し方で高品質なニューラル音声を提供します
