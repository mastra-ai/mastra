<div id="rerankwithscorer">
  # rerankWithScorer()
</div>

`rerankWithScorer()` 関数は、セマンティック関連性、ベクトル類似度、位置情報に基づくスコアリングを組み合わせることで、ベクトル検索結果を高度に再ランキングします。

```typescript
function rerankWithScorer({
  results: QueryResult[],
  query: string,
  scorer: RelevanceScoreProvider,
  options?: RerankerFunctionOptions,
}): Promise<RerankResult[]>;
```


<div id="usage-example">
  ## 使い方の例
</div>

```typescript
import { openai } from "@ai-sdk/openai";
import { rerankWithScorer as rerank, CohereRelevanceScorer } from "@mastra/rag";

const scorer = new CohereRelevanceScorer("rerank-v3.5");

const rerankedResults = await rerank({
  results: vectorSearchResults,
  query: "本番環境へのデプロイ方法は？",
  scorer,
  options: {
    weights: {
      semantic: 0.5,
      vector: 0.3,
      position: 0.2,
    },
    topK: 3,
  },
});
```

`rerankWithScorer` 関数は @mastra/rag の任意の `RelevanceScoreProvider` を受け取ります。

> 注: 再ランク付け時にセマンティック・スコアリングを正しく機能させるには、各結果の `metadata.text` フィールドにテキスト本文が含まれている必要があります。
