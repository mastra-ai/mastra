<div id="createdocumentchunkertool">
  # createDocumentChunkerTool()
</div>

`createDocumentChunkerTool()` 関数は、ドキュメントを小さなチャンクに分割し、効率的な処理と検索を可能にするツールを作成します。複数のチャンク手法と設定可能なパラメータに対応しています。

<div id="basic-usage">
  ## 基本的な使い方
</div>

```typescript
import { createDocumentChunkerTool, MDocument } from "@mastra/rag";

const document = new MDocument({
  text: "ドキュメントの内容をここに入力...",
  metadata: { source: "user-manual" },
});

const chunker = createDocumentChunkerTool({
  doc: document,
  params: {
    strategy: "recursive",
    size: 512,
    overlap: 50,
    separator: "\n",
  },
});

const { chunks } = await chunker.execute();
```

<div id="example-with-custom-parameters">
  ## カスタムパラメータを使った例
</div>

```typescript
const technicalDoc = new MDocument({
  text: longDocumentContent,
  metadata: {
    type: "technical",
    version: "1.0",
  },
});

const chunker = createDocumentChunkerTool({
  doc: technicalDoc,
  params: {
    strategy: "recursive",
    size: 1024, // チャンクを大きくする
    overlap: 100, // 重なりを増やす
    separator: "\n\n", // 空行（2連続の改行）で分割
  },
});

const { chunks } = await chunker.execute();

// チャンクを処理
chunks.forEach((chunk, index) => {
  console.log(`チャンク${index + 1}の長さ: ${chunk.content.length}`);
});
```


<div id="tool-details">
  ## ツールの詳細
</div>

このチャンクャーは、以下のプロパティを持つ Mastra ツールとして作成されます:

- **Tool ID**: `Document Chunker {strategy} {size}`
- **Description**: `{strategy} 戦略を用い、サイズ {size}、オーバーラップ {overlap} でドキュメントを分割`
- **Input Schema**: 空のオブジェクト（追加の入力は不要）
- **Output Schema**: チャンク配列を含むオブジェクト
