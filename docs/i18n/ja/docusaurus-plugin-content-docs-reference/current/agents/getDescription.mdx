# Agent.getDescription()

`.getDescription()` メソッドは、エージェントに設定された説明文を取得します。このメソッドは、エージェントの目的や機能を記したシンプルな文字列を返します。

## 使い方の例

```typescript copy
agent.getDescription();
```


## パラメータ

このメソッドはパラメータを取りません。

  content={[
    {
      name: "description",
      type: "string",
      description:
        "エージェントの説明。説明が設定されていない場合は空の文字列。",
    },
  ]}
