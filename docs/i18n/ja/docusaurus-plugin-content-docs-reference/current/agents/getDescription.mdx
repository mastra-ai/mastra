<div id="agentgetdescription">
  # Agent.getDescription()
</div>

`.getDescription()` メソッドは、エージェントに設定された説明文を取得します。このメソッドは、エージェントの目的や機能を記したシンプルな文字列を返します。

<div id="usage-example">
  ## 使い方の例
</div>

```typescript copy
agent.getDescription();
```


<div id="parameters">
  ## パラメータ
</div>

このメソッドはパラメータを取りません。

<div id="returns">
  ## 戻り値
</div>

<PropertiesTable
  content={[
    {
      name: "description",
      type: "string",
      description:
        "エージェントの説明。説明が設定されていない場合は空の文字列。",
    },
  ]}
/>
