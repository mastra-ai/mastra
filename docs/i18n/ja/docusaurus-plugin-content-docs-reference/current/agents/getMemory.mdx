# Agent.getMemory()

`.getMemory()` メソッドは、エージェントに関連付けられたメモリシステムを取得します。このメソッドは、会話をまたいで情報を保存・取得するためのエージェントのメモリ機能へアクセスする際に使用します。

## 使い方の例

```typescript copy
await agent.getMemory();
```
