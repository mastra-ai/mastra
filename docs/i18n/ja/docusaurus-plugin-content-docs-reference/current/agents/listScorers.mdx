<div id="agentlistscorers">
  # Agent.listScorers()
</div>

`.listScorers()` メソッドは、エージェントに設定されたスコアリング設定を取得し、それが関数である場合は評価して解決します。このメソッドにより、エージェントの応答やパフォーマンスを評価するために用いられるスコアリングシステムにアクセスできます。

<div id="usage-example">
  ## 使い方の例
</div>

```typescript copy
await agent.listScorers();
```
