<div id="agentlistworkflows">
  # Agent.listWorkflows()
</div>

`.listWorkflows()` メソッドは、エージェントに設定されたワークフローを取得し、関数として指定されている場合はそれを解決します。これらのワークフローにより、エージェントは定義済みの実行パスに従って、複雑なマルチステップの処理を実行できます。

<div id="usage-example">
  ## 使い方の例
</div>

```typescript copy
await agent.listWorkflows();
```
