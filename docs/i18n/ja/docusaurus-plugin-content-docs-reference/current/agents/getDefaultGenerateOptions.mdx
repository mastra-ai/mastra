<div id="agentgetdefaultgenerateoptions">
  # Agent.getDefaultGenerateOptions()
</div>

エージェントは、モデルの挙動、出力のフォーマット、ツールやワークフローの呼び出しを制御するためのデフォルト生成オプションを設定できます。`.getDefaultGenerateOptions()` メソッドは、これらのデフォルトを取得し、関数であれば評価して解決します。これらのオプションは、上書きされない限りすべての `generate()` 呼び出しに適用され、エージェントの既定設定を確認するのに役立ちます。

<div id="usage-example">
  ## 使い方の例
</div>

```typescript copy
await agent.getDefaultGenerateOptions();
```
