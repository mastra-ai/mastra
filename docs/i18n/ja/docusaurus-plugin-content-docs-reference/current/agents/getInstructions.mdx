<div id="agentgetinstructions">
  # Agent.getInstructions()
</div>

`.getInstructions()` メソッドは、エージェントに設定された指示を取得し、それが関数の場合は評価して解決します。これらの指示はエージェントの動作を導き、その能力と制約を定義します。

<div id="usage-example">
  ## 使い方の例
</div>

```typescript copy
await agent.getInstructions();
```
