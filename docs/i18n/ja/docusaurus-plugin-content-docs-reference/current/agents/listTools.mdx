<div id="agentlisttools">
  # Agent.listTools()
</div>

`.listTools()` メソッドは、エージェントに設定されたツールを取得し、関数である場合はそれを解決します。これらのツールはエージェントの機能を拡張し、特定の操作の実行や外部システムへのアクセスを可能にします。

<div id="usage-example">
  ## 使い方の例
</div>

```typescript copy
await agent.listTools();
```
