<div id="memorygetthreadbyid">
  # Memory.getThreadById()
</div>

`.getThreadById()` メソッドは、ID を指定して特定のスレッドを取得します。

<div id="usage-example">
  ## 使い方の例
</div>

```typescript
await memory?.getThreadById({ threadId: "thread-123" });
```

  ## パラメータ
</div>

<PropertiesTable
  content={[
    {
      name: "threadId",
      type: "string",
      description: "取得対象のスレッド ID。",
      isOptional: false,
    },
  ]}
/>
