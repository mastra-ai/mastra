<div id="braintrustexporter">
  # BraintrustExporter
</div>

評価や可観測性のために、AI のトレースデータを Braintrust に送信します。

<div id="constructor">
  ## コンストラクタ
</div>

```typescript
new BraintrustExporter(config: BraintrustExporterConfig)
```


<div id="braintrustexporterconfig">
  ## BraintrustExporterConfig
</div>

```typescript
interface BraintrustExporterConfig {
  apiKey?: string;
  endpoint?: string;
  projectName?: string;
  logLevel?: "debug" | "info" | "warn" | "error";
  tuningParameters?: Record<string, any>;
}
```

```typescript
async exportEvent(event: AITracingEvent): Promise<void>
```

```typescript
async export(spans: ReadOnlyAISpan[]): Promise<void>
```

  ### シャットダウン
</div>

```typescript
async shutdown(): Promise<void>
```

<div id="usage">
  ## 使い方
</div>

```typescript
import { BraintrustExporter } from "@mastra/braintrust";

const exporter = new BraintrustExporter({
  apiKey: process.env.BRAINTRUST_API_KEY,
  projectName: "my-ai-project",
});
```

  ## スパンタイプのマッピング

| `TOOL_CALL`                 | `tool`            |
| `MCP_TOOL_CALL`             | `tool`            |
| `WORKFLOW_CONDITIONAL_EVAL` | `function`        |
| `WORKFLOW_WAIT_EVENT`       | `function`        |
| その他                      | `task`            |
