<div id="new-relic">
  # New Relic
</div>

New Relic は、フルスタック監視のために OpenTelemetry（OTLP）をサポートする、包括的なオブザーバビリティ・プラットフォームです。

<div id="configuration">
  ## 設定
</div>

OTLP 経由で Mastra と併用して New Relic を利用するには、次の環境変数を設定してください。

```env
OTEL_EXPORTER_OTLP_ENDPOINT=https://otlp.nr-data.net:4317
OTEL_EXPORTER_OTLP_HEADERS="api-key=your_license_key"
```


<div id="implementation">
  ## 実装
</div>

Mastra を New Relic を使って利用するように設定する方法は次のとおりです。

```typescript
import { Mastra } from "@mastra/core";

export const mastra = new Mastra({
  // ... その他の設定
  telemetry: {
    serviceName: "your-service-name",
    enabled: true,
    export: {
      type: "otlp",
    },
  },
});
```
