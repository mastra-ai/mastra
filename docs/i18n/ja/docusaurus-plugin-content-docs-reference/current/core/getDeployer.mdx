<div id="mastragetdeployer">
  # Mastra.getDeployer()
</div>

`.getDeployer()` メソッドは、Mastra インスタンスで設定されたデプロイアインスタンスを取得するために使用します。

<div id="usage-example">
  ## 使い方の例
</div>

```typescript copy
mastra.getDeployer();
```


<div id="parameters">
  ## パラメータ
</div>

このメソッドはパラメータを受け付けません。
