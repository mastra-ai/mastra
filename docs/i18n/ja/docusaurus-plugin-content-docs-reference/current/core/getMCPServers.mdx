<div id="mastragetmcpservers">
  # Mastra.getMCPServers()
</div>

`.getMCPServers()` メソッドは、Mastra インスタンスに登録されているすべての MCP サーバーインスタンスを取得します。

<div id="usage-example">
  ## 使い方の例
</div>

```typescript copy
mastra.getMCPServers();
```


<div id="parameters">
  ## パラメータ
</div>

このメソッドはパラメータを受け取りません。

<div id="returns">
  ## 戻り値
</div>

<PropertiesTable
  content={[
    {
      name: "servers",
      type: "Record<string, MCPServerBase> | undefined",
      description:
        "登録済みのすべての MCP サーバーインスタンスを保持するレコード。キーはサーバー ID、値は MCPServerBase インスタンスで、サーバーが一切登録されていない場合は undefined になります。",
    },
  ]}
/>
