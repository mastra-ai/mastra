<div id="mastragetlogs">
  # Mastra.getLogs()
</div>

`.getLogs()` メソッドは、特定の transport ID に紐づくすべてのログを取得するために使用します。このメソッドを使用するには、`getLogs` 操作をサポートするように構成されたロガーが必要です。

<div id="usage-example">
  ## 使い方の例
</div>

```typescript copy
mastra.getLogs("456");
```
