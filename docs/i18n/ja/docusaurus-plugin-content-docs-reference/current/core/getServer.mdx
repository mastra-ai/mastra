<div id="mastragetserver">
  # Mastra.getServer()
</div>

`.getServer()` メソッドは、Mastra インスタンスで設定されているサーバー設定を取得するために使用します。

<div id="usage-example">
  ## 使い方の例
</div>

```typescript copy
mastra.getServer();
```


<div id="parameters">
  ## パラメータ
</div>

このメソッドはパラメータを受け付けません。

  ## 戻り値
</div>

    },
  ]}
