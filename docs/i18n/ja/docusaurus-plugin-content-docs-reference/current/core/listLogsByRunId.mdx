<div id="mastralistlogsbyrunid">
  # Mastra.listLogsByRunId()
</div>

`.listLogsByRunId()` メソッドは、特定の実行 ID とトランスポート ID に対応するログを取得するために使用します。このメソッドを利用するには、`listLogsByRunId` 操作をサポートするように構成されたロガーが必要です。

<div id="usage-example">
  ## 使い方の例
</div>

```typescript copy
mastra.listLogsByRunId({ runId: "123", transportId: "456" });
```
