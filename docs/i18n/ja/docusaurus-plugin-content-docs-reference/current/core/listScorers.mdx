<div id="listscorers">
  # listScorers()
</div>

`listScorers()` メソッドは、Mastra インスタンスに登録されているすべてのスコアラーを返します。スコアラーは AI の出力を評価するために使用され、エージェントの生成やワークフローの実行時に、デフォルトのスコアラーを上書きできます。

<div id="usage-example">
  ## 使い方の例
</div>

```typescript
import { mastra } from "./mastra";

// 登録されているすべてのスコアラーを取得
const allScorers = mastra.listScorers();

// 特定のスコアラーにアクセス
const myScorer = allScorers.relevancyScorer;
```


<div id="parameters">
  ## パラメータ
</div>

このメソッドはパラメータを受け取りません。
