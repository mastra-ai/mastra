<div id="mastragetvectors">
  # Mastra.getVectors()
</div>

`.getVectors()` メソッドは、Mastra インスタンスで設定されたすべてのベクターストアを取得するために使用します。

<div id="usage-example">
  ## 使用例
</div>

```typescript copy
mastra.getVectors();
```


<div id="parameters">
  ## パラメータ
</div>

このメソッドはパラメータを受け取りません。
