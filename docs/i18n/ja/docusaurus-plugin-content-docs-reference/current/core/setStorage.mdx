<div id="mastrasetstorage">
  # Mastra.setStorage()
</div>

`.setStorage()` メソッドは、Mastra インスタンスに対して使用するストレージインスタンスを設定します。引数には `MastraStorage` を1つ渡します。

<div id="usage-example">
  ## 使い方の例
</div>

```typescript copy
mastra.setStorage(
  new LibSQLStore({
    url: ":memory:",
  }),
);
```
