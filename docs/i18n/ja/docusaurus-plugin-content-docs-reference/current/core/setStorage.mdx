# Mastra.setStorage()

`.setStorage()` メソッドは、Mastra インスタンスに対して使用するストレージインスタンスを設定します。引数には `MastraStorage` を1つ渡します。

## 使い方の例

```typescript copy
mastra.setStorage(
  new LibSQLStore({
    url: ":memory:",
  }),
);
```
