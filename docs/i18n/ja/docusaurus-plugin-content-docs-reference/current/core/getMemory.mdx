<div id="mastragetmemory">
  # Mastra.getMemory()
</div>

`.getMemory()` メソッドは、Mastra インスタンスで構成されたメモリ インスタンスを取得するために使用します。

<div id="usage-example">
  ## 使用例
</div>

```typescript copy
mastra.getMemory();
```


<div id="parameters">
  ## パラメータ
</div>

このメソッドはパラメータを受け取りません。

  ## 戻り値
</div>

    },
  ]}
