<div id="mastragettelemetry">
  # Mastra.getTelemetry()
</div>

`.getTelemetry()` メソッドは、Mastra インスタンスで構成されたテレメトリ インスタンスを取得するために使用します。

<div id="usage-example">
  ## 使い方の例
</div>

```typescript copy
mastra.getTelemetry();
```


<div id="parameters">
  ## パラメータ
</div>

このメソッドはパラメータを受け取りません。

</div>

    },
  ]}
