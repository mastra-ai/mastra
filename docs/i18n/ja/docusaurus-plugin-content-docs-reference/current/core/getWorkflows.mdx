<div id="mastragetworkflows">
  # Mastra.getWorkflows()
</div>

`.getWorkflows()` メソッドは、Mastra インスタンスで設定されたすべてのワークフローを取得します。オプションとして options オブジェクトを渡せます。

<div id="usage-example">
  ## 使い方の例
</div>

```typescript copy
mastra.getWorkflows();
```

      name: "options",
      type: "{ serialized?: boolean }",
      description:

      optional: true,
    },
  ]}
