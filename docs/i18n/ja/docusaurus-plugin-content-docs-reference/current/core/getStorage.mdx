<div id="mastragetstorage">
  # Mastra.getStorage()
</div>

`.getStorage()` メソッドは、Mastra インスタンスで設定されたストレージインスタンスを取得するために使用します。

<div id="usage-example">
  ## 使い方の例
</div>

```typescript copy
mastra.getStorage();
```


<div id="parameters">
  ## パラメータ
</div>

このメソッドはパラメータを受け取りません。
