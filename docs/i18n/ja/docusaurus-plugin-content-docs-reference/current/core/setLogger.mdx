<div id="mastrasetlogger">
  # Mastra.setLogger()
</div>

`.setLogger()` メソッドは、Mastra インスタンス内のすべてのコンポーネント（エージェント、ワークフローなど）のロガーを設定します。このメソッドは、logger プロパティを持つ単一のオブジェクトを引数として受け取ります。

<div id="usage-example">
  ## 使い方の例
</div>

```typescript copy
mastra.setLogger({ logger: new PinoLogger({ name: "testLogger" }) });
```
