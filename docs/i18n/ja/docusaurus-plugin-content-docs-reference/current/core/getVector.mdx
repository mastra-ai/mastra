<div id="mastragetvector">
  # Mastra.getVector()
</div>

`.getVector()` メソッドは、ベクターストアを名前で取得するためのものです。引数として、ベクターストア名を表す `string` を1つ受け取ります。

<div id="usage-example">
  ## 使用例
</div>

```typescript copy
mastra.getVector("testVectorStore");
```


<div id="parameters">
  ## パラメータ
</div>

<PropertiesTable
  content={[
    {
      name: "name",
      type: "TVectorName extends keyof TVectors",
      description:
        "取得するベクターストアの名前。Mastra の構成に存在する有効なベクターストア名である必要があります。",
    },
  ]}
/>

  ## 戻り値

    },
  ]}
/>
