<div id="mastralistlogs">
  # Mastra.listLogs()
</div>

`.listLogs()` メソッドは、特定の transport ID に対するすべてのログを取得するために使用します。このメソッドを利用するには、`listLogs` 操作に対応した設定済みのロガーが必要です。

<div id="usage-example">
  ## 使用例
</div>

```typescript copy
mastra.listLogs("456");
```
