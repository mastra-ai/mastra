<div id="mastragetlogger">
  # Mastra.getLogger()
</div>

`.getLogger()` メソッドは、Mastra インスタンスで構成されたロガーインスタンスを取得するために使用します。

<div id="usage-example">
  ## 使い方の例
</div>

```typescript copy
mastra.getLogger();
```


<div id="parameters">
  ## パラメータ
</div>

このメソッドはパラメータを受け付けません。

<div id="returns">
  ## 戻り値
</div>

<PropertiesTable
  content={[
    {
      name: "logger",
      type: "TLogger",
      description:
        "すべてのコンポーネント（エージェント、ワークフローなど）でのログ出力に使用される、設定済みのロガーインスタンス。",
    },
  ]}
/>
