<div id="mastragetagents">
  # Mastra.getAgents()
</div>

`.getAgents()` メソッドは、Mastra インスタンスで設定されているすべてのエージェントを取得するために使用します。

<div id="usage-example">
  ## 使い方の例
</div>

```typescript copy
mastra.getAgents();
```


<div id="parameters">
  ## パラメータ
</div>

このメソッドはパラメータを受け取りません。
