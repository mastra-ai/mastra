<div id="runcancel">
  # Run.cancel()
</div>

`.cancel()` メソッドはワークフロー実行をキャンセルし、処理を停止してリソースを解放します。

<div id="usage-example">
  ## 使い方の例
</div>

```typescript showLineNumbers copy
const run = await workflow.createRunAsync();

await run.cancel();
```

  ## 発展的な使用例
</div>

```typescript showLineNumbers copy
const run = await workflow.createRunAsync();

try {
  const result = await run.start({ inputData: { value: "初期データ" } });
} catch (error) {
  await run.cancel();
}
```
