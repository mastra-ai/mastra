<div id="workflowthen">
  # Workflow.then()
</div>

`.then()` メソッドは、ワークフローの各ステップ間に順次の依存関係を設定し、ステップが所定の順序で実行されるようにします。

<div id="usage-example">
  ## 使用例
</div>

```typescript copy
workflow.then(step1).then(step2);
```
