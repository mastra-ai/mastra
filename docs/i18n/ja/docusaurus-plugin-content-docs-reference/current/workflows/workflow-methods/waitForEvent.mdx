<div id="workflowwaitforevent">
  # Workflow.waitForEvent()
</div>

`.waitForEvent()` メソッドは、イベントを受信するまで実行を一時停止します。

<div id="usage-example">
  ## 使い方の例
</div>

```typescript copy
workflow.waitForEvent("event-name", step1);
```

  ## パラメータ

    },

      name: "step",
      type: "Step",
      description: "イベント受信後に再開するステップ",
      isOptional: false,

/>

<div id="extended-usage-example">
  ## 応用的な使用例
</div>

```typescript showLineNumbers copy
import { createWorkflow, createStep } from "@mastra/core/workflows";

const step1 = createStep({...});
const step2 = createStep({...});
const step3 = createStep({...});

export const testWorkflow = createWorkflow({...})
  .then(step1)
  .waitForEvent("event-name", step2)
  .then(step3)
  .commit();
```
