<div id="workflowbranch">
  # Workflow.branch()
</div>

`.branch()` メソッドは、ワークフローのステップ間に条件分岐を作成し、直前のステップの結果に応じて異なる処理経路を選択できるようにします。

<div id="usage-example">
  ## 使い方の例
</div>

```typescript copy
workflow.branch([
  [async ({ context }) => true, step1],
  [async ({ context }) => false, step2],
]);
```
