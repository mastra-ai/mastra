<div id="workflowmap">
  # Workflow.map()
</div>

`.map()` メソッドは、前のステップの出力データを後続のステップの入力に対応付け、ステップ間でデータを変換できるようにします。

<div id="usage-example">
  ## 使い方の例
</div>

```typescript copy
workflow.map(async ({ inputData }) => `${inputData.value} - map`
```
