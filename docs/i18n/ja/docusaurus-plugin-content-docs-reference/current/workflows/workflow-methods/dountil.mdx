<div id="workflowdountil">
  # Workflow.dountil()
</div>

`.dountil()` メソッドは、条件が満たされるまでステップを実行します。条件を評価する前に、必ずステップを少なくとも一度実行します。条件が初めて評価されるとき、`iterationCount` は `1` です。

<div id="usage-example">
  ## 使い方の例
</div>

```typescript copy
workflow.doUntil(step1, async ({ inputData }) => true);
```
