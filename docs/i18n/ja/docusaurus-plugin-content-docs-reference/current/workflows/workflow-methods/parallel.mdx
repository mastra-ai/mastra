<div id="workflowparallel">
  # Workflow.parallel()
</div>

`.parallel()` メソッドは複数のステップを並列に実行します。

<div id="usage-example">
  ## 使い方の例
</div>

```typescript copy
workflow.parallel([step1, step2]);
```
