<div id="workflowforeach">
  # Workflow.foreach()
</div>

`.foreach()` メソッドは、配列内の各要素ごとにステップを実行するループを作成します。

<div id="usage-example">
  ## 使い方の例
</div>

```typescript copy
workflow.foreach(step1, { concurrency: 2 });
```
