# run.watch()

`.watch()` 関数は、mastra の実行（run）の状態変化を購読し、実行の進捗を監視して状態更新に応答できるようにします。

## 使い方の例

```typescript
import { LegacyWorkflow } from "@mastra/core/workflows/legacy";

const workflow = new LegacyWorkflow({
  name: "document-processor",
});

const run = workflow.createRun();

// Subscribe to state changes
const unsubscribe = run.watch(({ results, activePaths }) => {
  console.log("結果:", results);
  console.log("アクティブなパス:", activePaths);
});

// 状態変更を監視する
await run.start({
  input: { text: "このドキュメントを処理してください" },
});

// ワークフローを実行
// 監視を停止
```

## 追加例

特定のステップの完了を監視する:

```typescript
run.watch(({ results, activePaths }) => {
  if (activePaths.get("processDocument")?.status === "completed") {
    console.log(
      "ドキュメント処理の出力：",
      results["processDocument"].output,
    );
  }
});
```

エラー処理:

```typescript
run.watch(({ results, activePaths }) => {
  if (activePaths.get("processDocument")?.status === "failed") {
    console.error(
      "ドキュメント処理に失敗しました:",
      results["processDocument"].error,
    );
    // エラー復旧ロジックを実装する
  }
});
```
