<div id="workflowcommit">
  # Workflow.commit()
</div>

`.commit()` メソッドは、現在のステップ構成を用いてワークフローのステートマシンを再初期化します。

<div id="usage">
  ## 使い方
</div>

```typescript
workflow.step(stepA).then(stepB).commit();
```
