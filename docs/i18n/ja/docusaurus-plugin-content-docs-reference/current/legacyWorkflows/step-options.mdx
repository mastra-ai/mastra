<div id="stepoptions">
  # StepOptions
</div>

変数マッピング、実行条件、その他のランタイム動作を制御するワークフローの各ステップ用の設定オプション。

<div id="usage">
  ## 使い方
</div>

```typescript
workflow.step(processOrder, {
  variables: {
    orderId: { step: "trigger", path: "id" },
    userId: { step: "auth", path: "user.id" },
  },
  when: {
    ref: { step: "auth", path: "status" },
    query: { $eq: "認証済" },
  },
});
```
