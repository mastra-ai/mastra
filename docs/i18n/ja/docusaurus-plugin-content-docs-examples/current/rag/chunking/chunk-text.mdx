  # チャンクテキスト
</div>

大きなテキストドキュメントを扱う際は、処理しやすい小さな単位に分割する必要があります。チャンク手法は、検索・分析・取得に利用できるセグメントにテキストコンテンツを分割します。この例では、デフォルト設定でプレーンテキストをチャンクに分割する方法を示します。

```tsx copy
import { MDocument } from "@mastra/rag";

const doc = MDocument.fromText("プレーンテキストのコンテンツ…");

const chunks = await doc.chunk();
```
