<div id="smart-voice-memo-app">
  # スマート音声メモアプリ
</div>

以下のコードスニペットは、Next.js と Mastra を直接統合したスマート音声メモアプリにおける Speech-to-Text（STT）機能の実装例です。Next.js への Mastra の統合について詳しくは、[Next.js と統合する](/docs/frameworks/web-frameworks/next-js)のドキュメントをご覧ください。

<div id="creating-an-agent-with-stt-capabilities">
  ## STT 機能を備えたエージェントの作成
</div>

次の例では、OpenAI の STT 機能を用いて音声対応エージェントを初期化する方法を紹介します。

```typescript title="src/mastra/agents/index.ts"
import { openai } from "@ai-sdk/openai";
import { Agent } from "@mastra/core/agent";
import { OpenAIVoice } from "@mastra/voice-openai";

const instructions = `
あなたは、コンテンツの簡潔で構造化された要約を提供するAIノートアシスタントです... // 簡潔化のため省略
`;

export const noteTakerAgent = new Agent({
  name: "ノート作成エージェント",
  instructions: instructions,
  model: openai("gpt-4o"),
  voice: new OpenAIVoice(), // デフォルト設定でOpenAI音声プロバイダを追加
});
```


<div id="registering-the-agent-with-mastra">
  ## Mastra へのエージェントの登録
</div>

このスニペットは、STT 対応のエージェントをお使いの Mastra インスタンスに登録する方法を示します。

```typescript title="src/mastra/index.ts"
import { PinoLogger } from "@mastra/loggers";
import { Mastra } from "@mastra/core/mastra";

import { noteTakerAgent } from "./agents";

export const mastra = new Mastra({
  agents: { noteTakerAgent }, // ノートテイカーエージェントを登録
  logger: new PinoLogger({
    name: "Mastra",
    level: "info",
  }),
});
```


<div id="processing-audio-for-transcription">
  ## 文字起こしのための音声処理
</div>

次のコードは、Web リクエストから音声を受け取り、エージェントの STT 機能を使って文字起こしする方法を示します:

```typescript title="app/api/audio/route.ts"
import { mastra } from "@/src/mastra"; // Mastraインスタンスをインポート
import { Readable } from "node:stream";

export async function POST(req: Request) {
  // リクエストから音声ファイルを取得
  const formData = await req.formData();
  const audioFile = formData.get("audio") as File;
  const arrayBuffer = await audioFile.arrayBuffer();
  const buffer = Buffer.from(arrayBuffer);
  const readable = Readable.from(buffer);

  // Mastraインスタンスからノートテイカーエージェントを取得
  const noteTakerAgent = mastra.getAgent("noteTakerAgent");

  // 音声ファイルを文字起こし
  const text = await noteTakerAgent.voice?.listen(readable);

  return new Response(JSON.stringify({ text }), {
    headers: { "Content-Type": "application/json" },
  });
}
```
