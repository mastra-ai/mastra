<div id="discord-community">
  # Discord コミュニティ
</div>

Discord サーバーには 1,000 人以上のメンバーが参加しており、Mastra の主要なディスカッションフォーラムとして機能しています。Mastra チームは北米および欧州の営業時間内に Discord を常時モニタリングしており、その他のタイムゾーンでもコミュニティ メンバーが活動しています。[Discord サーバーに参加](https://discord.gg/BTYqqHKUrf)。

<div id="discord-mcp-bot">
  ## Discord MCP Bot
</div>

コミュニティメンバーに加えて、質問に答える手助けをしてくれる（実験的な）Discordボットも用意しています。これは [Model Context Protocol (MCP)](/docs/tools-mcp/mcp-overview) を使用しています。公開チャンネルまたはDMで `/ask` を使って質問でき、DMでは `/cleardm` で履歴を消去できます。
