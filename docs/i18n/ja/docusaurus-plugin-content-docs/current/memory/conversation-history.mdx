<div id="conversation-history">
  # 会話履歴
</div>

会話履歴は最も基本的なタイプのメモリです。現在の会話に含まれるメッセージのリストを指します。

デフォルトでは、各リクエストに現在のメモリスレッドから直近のメッセージ10件が含まれ、エージェントに短期的な会話コンテキストを提供します。この上限は `lastMessages` パラメータで引き上げることができます。

この上限は、`Memory` インスタンスに `lastMessages` パラメータを渡すことで引き上げられます。

```typescript {3-7} showLineNumbers
export const testAgent = new Agent({
  // ...
  memory: new Memory({
    options: {
      lastMessages: 20,
    },
  }),
});
```
