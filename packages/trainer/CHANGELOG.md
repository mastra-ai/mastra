# @mastra/trainer

## 0.1.0-alpha.1

### Features

- Initial release of the Mastra Trainer package
- Support for SFT (Supervised Fine-Tuning), DPO (Direct Preference Optimization), and RFT (Reinforcement Fine-Tuning)
- Dataset sources: traces from observability storage, arrays, and JSONL files
- Composite scoring with weighted averages and gates
- JSONL rendering for OpenAI fine-tuning format
- OpenAI provider for fine-tuning jobs
- Selection utilities: deduplication, balancing, holdout splits
