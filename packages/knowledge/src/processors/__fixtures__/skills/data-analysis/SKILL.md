# Data Analysis Skill

Provides comprehensive data analysis capabilities.

## Capabilities

- Load data from CSV, Excel, JSON formats
- Perform statistical analysis
- Generate visualizations (charts, graphs, plots)
- Create summary reports
- Detect patterns and anomalies

## Quick Start

1. Load your dataset using the appropriate loader
2. Explore the data with summary statistics
3. Create visualizations to understand patterns
4. Generate insights and reports

## Best Practices

- Always check for missing values first
- Validate data types before analysis
- Use appropriate chart types for your data
- Document your analysis steps
