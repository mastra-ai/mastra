Add MnemoPay MCP example — AI agent with economic memory

New example at `examples/mcp-mnemopay` demonstrating MnemoPay integration via MCP.
Agent gets 12 tools automatically: memory operations (remember, recall, forget,
reinforce, consolidate) and economic operations (charge, settle, refund, balance,
profile, logs, history). Uses published `@mnemopay/sdk` from npm via npx.
