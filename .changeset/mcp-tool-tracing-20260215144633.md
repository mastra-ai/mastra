Added optional observability support to MCP tool execution. Pass a `tracingContext` when calling a tool to enable automatic span creation for each tool invocation.

```ts
const tools = await mcpClient.tools();

const result = await tools.myTool.execute(input, {
  tracingContext,
});
```

When `tracingContext` is provided, each tool call is traced with input/output data and success/failure status. If no `tracingContext` is passed, behavior is unchanged.
