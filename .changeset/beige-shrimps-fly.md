Added `create-agentbuilder` package — run `npm create agentbuilder` to scaffold a new Mastra project with Agent Builder pre-configured, Observability opt-in, and sensible defaults. The CLI prompts for a project name and Observability auth, then generates a ready-to-run project with the Agent Builder mounted at `localhost:4111/agent-builder`.
