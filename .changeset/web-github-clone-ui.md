Added the GitHub repo flow to the MastraCode web UI. You can now open a repository from GitHub directly in the projects dialog: pick a repo, and it is cloned into an isolated cloud sandbox with live progress shown while the sandbox is provisioned and the repo is cloned. If preparation fails, the previous project stays active and the error is shown so you can retry. GitHub projects now also display their repo name, branch, and sandbox path throughout the app.
