Updated Mastra Studio font bundle. Studio now uses Inter for both display and body text, shipped as a 352KB variable `.woff2` (down from the 862KB Inter TTF). Code now uses Commit Mono (regular + bold `.woff2`) instead of Geist Mono. The legacy TasaExplorer display font has been removed. All bundled fonts are SIL OFL licensed. All faces use `font-display: swap`. Net Studio font payload: ~948KB → ~489KB.
